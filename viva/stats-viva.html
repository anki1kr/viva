Program Six · Distribution Shape

Skewness measure karna

🎯 Objective

Data symmetric hai ya ek taraf zyada jhuka hua hai — ye skewness se pata chalta hai. Excel ke SKEW() function se calculate karna.

💡 Concept

Skewness Ke Types

  • Skewness = 0 → Data perfectly symmetric (normal distribution)
  • Skewness {">"} 0 (Positive) → Right-skewed, tail right side mai lambi
  • Skewness {"<"} 0 (Negative) → Left-skewed, tail left side mai lambi
  • Range → -1 se 1 tak ho toh roughly symmetric, beyond means highly skewed

Result Interpret Karo

Agar value positive (jaise 1.85) hai → right-skewed (data ke bade values right mai pull kar rahe hai). Agar negative hai → left-skewed.

Bonus: Mean, Median, Mode bhi Nikalo

💡 Concept

Linear Regression Kya Hai?

  • Y (Dependent) — wo variable jo predict karna hai (jaise marks)

FunctionUse

Correlation Coefficient calculate karna

🎯 Objective

Do variables ke beech kitna strong relationship hai — wo correlation coefficient (r) se measure karna. Excel ke CORREL() function se.

Correlation Coefficient (r)

  • r = +1 → Perfect positive correlation (ek badhe toh dusra bhi badhe)

Probability & Random Variables

🔥 HotDefine a random variable. What are its types?
A random variable is a function that assigns a numerical value to each outcome of a random experiment. Types: Discrete (countable values, e.g. number of heads in 5 tosses) and Continuous (any value in an interval, e.g. height, weight).
What are the rules of probability? State the sum and product rules.
Sum rule (Addition): P(A∪B) = P(A) + P(B) − P(A∩B). For mutually exclusive events: P(A∪B) = P(A) + P(B).
Product rule (Multiplication): P(A∩B) = P(A) · P(B|A). For independent events: P(A∩B) = P(A) · P(B).
What is joint probability?
Joint probability P(A∩B) is the probability that both events A and B occur simultaneously. It is found using the multiplication rule: P(A∩B) = P(A) × P(B|A).

PMF, PDF and CDF

★ FavWhat is the difference between PMF and PDF?
PMF (Probability Mass Function) is for discrete random variables — gives P(X = x) for each value x. All values sum to 1.
PDF (Probability Density Function) is for continuous random variables — P(X = x) = 0 for any single point; instead P(a ≤ X ≤ b) = ∫ f(x)dx from a to b. Total area under curve = 1.
State the properties of CDF (Cumulative Distribution Function).
CDF: F(x) = P(X ≤ x). Properties:
  • 0 ≤ F(x) ≤ 1 for all x
  • F(−∞) = 0, F(+∞) = 1
  • F(x) is non-decreasing
  • F(x) is right-continuous
  • P(a < X ≤ b) = F(b) − F(a)

Discrete Distributions

🔥 HotDefine Binomial distribution. When is it used? Give its mean and variance.
Binomial distribution models the number of successes in n independent Bernoulli trials, each with probability p of success.
Used when: fixed n, two outcomes, constant p, independent trials.
P(X = x) = ⁿCₓ · pˣ · (1−p)ⁿ⁻ˣ, x = 0,1,…,n Mean = np Variance = np(1−p)
🔥 HotDefine Poisson distribution. When is it used? Why is mean = variance?
Poisson distribution models the number of events in a fixed interval when events occur independently at a constant average rate λ.
Conditions: rare events, large n, small p, np = λ (finite).
P(X = x) = e⁻λ · λˣ / x!, x = 0,1,2,… Mean = λ, Variance = λ (always equal)
Mean = variance because the MGF derivation yields E[X] = λ and Var(X) = λ identically.
What is Geometric distribution? What is its memoryless property?
Models the number of trials until the first success. P(X = x) = (1−p)ˣ⁻¹ · p, x = 1,2,…
Memoryless property: P(X > m+n | X > m) = P(X > n) — the past has no effect on future trials.

Continuous Distributions

🔥 HotState the properties of Normal distribution and the 68-95-99.7 rule.
Normal distribution N(μ, σ²): bell-shaped, symmetric about mean μ, fully described by μ and σ.
f(x) = (1/σ√2π) · e^(−(x−μ)²/2σ²) 68% of data lies within μ ± 1σ 95% of data lies within μ ± 2σ 99.7% of data lies within μ ± 3σ
Mean = Median = Mode = μ. Total area under curve = 1.
What is Exponential distribution? How is it related to Poisson?
Models the waiting time between Poisson events. f(x) = λe⁻λˣ, x ≥ 0. Mean = 1/λ, Variance = 1/λ². Also memoryless. If events follow Poisson(λ), inter-arrival times follow Exponential(λ).
What is Uniform distribution? When is it used?
All outcomes in [a, b] are equally likely. f(x) = 1/(b−a) for a ≤ x ≤ b. Mean = (a+b)/2, Variance = (b−a)²/12. Used when no outcome is preferred — e.g. random number generators.
Discrete Distributions
  • Binomial — n trials, p success
  • Poisson — rare events, rate λ
  • Geometric — trials until 1st success
Continuous Distributions
  • Normal — bell curve, μ & σ
  • Exponential — waiting time
  • Uniform — equal likelihood in [a,b]
II

Moments & Moment Generating Functions

Expectation · MGF · PGF · 2D Random Variables · Joint / Marginal / Conditional

Expectation & Moments

🔥 HotDefine the Expectation (mean) of a random variable.
The expected value E[X] is the weighted average of all possible values.
Discrete: E[X] = Σ x · P(X = x) Continuous: E[X] = ∫ x · f(x) dx
It represents the long-run average value of the variable.
What is the formula for Variance in terms of Expectation?
Var(X) = E[X²] − (E[X])² = E[(X − μ)²] where μ = E[X]
Variance measures the spread/dispersion of values around the mean.
What are the rth moments about the origin and about the mean?
rth moment about origin (raw moment): μ'ᵣ = E[Xʳ]
rth central moment (about mean): μᵣ = E[(X − μ)ʳ]
1st central moment = 0. 2nd central moment = Variance. 3rd → skewness. 4th → kurtosis.

MGF & PGF

🔥 HotWhat is the Moment Generating Function (MGF)? Why is it useful?
M_X(t) = E[e^(tX)] = Σ or ∫ e^(tx) · f(x)
Why useful: The rth derivative of M_X(t) at t=0 gives the rth moment: E[Xʳ] = M_X⁽ʳ⁾(0). It uniquely identifies a distribution — two distributions with the same MGF are identical.
State the properties of MGF.
  • M_X(0) = 1 always
  • E[Xʳ] = d^r/dt^r M_X(t) |t=0
  • If X, Y independent: M_{X+Y}(t) = M_X(t) · M_Y(t)
  • MGF uniquely determines the distribution (if it exists in a neighbourhood of 0)
What is the Probability Generating Function (PGF)?
PGF is defined for non-negative integer-valued discrete RVs:
G_X(z) = E[z^X] = Σ P(X=k) · zᵏ
Differentiating at z=1 gives factorial moments. G'(1) = E[X], G''(1) = E[X(X−1)]. Useful for sums of independent discrete RVs.

Two-Dimensional Random Variables

🔥 HotWhat is a joint distribution? How do you get marginal distributions from it?
Joint distribution f(x,y) gives the probability of X and Y taking values simultaneously.
Discrete: P(X=x, Y=y) = f(x,y) Marginal of X: f_X(x) = Σ_y f(x,y) Marginal of Y: f_Y(y) = Σ_x f(x,y)
What is a conditional distribution? Give the formula.
Conditional distribution of X given Y = y:
f(x|y) = f(x,y) / f_Y(y), provided f_Y(y) > 0
It describes how X is distributed when Y is fixed at y.
★ FavWhen are two random variables independent?
X and Y are independent iff their joint distribution factors into the product of marginals:
f(x,y) = f_X(x) · f_Y(y) for all x, y
Equivalently: f(x|y) = f_X(x) — knowing Y gives no information about X.
What is Covariance? How is it computed using Expectation?
Cov(X,Y) = E[XY] − E[X]·E[Y] = E[(X−μ_X)(Y−μ_Y)]
Positive covariance → X and Y increase together. Negative → one increases as other decreases. If X,Y independent → Cov(X,Y) = 0 (but zero cov does not imply independence).
III

Descriptive Statistics

Types of Data · Frequency Distribution · Central Tendency · Skewness · Kurtosis

Types of Data & Classification

🔥 HotWhat is the difference between primary and secondary data?
Primary Data
  • Collected first-hand by the researcher
  • Original, fresh, specific to the purpose
  • Examples: surveys, experiments, interviews
  • More accurate but time-consuming
Secondary Data
  • Already collected by someone else
  • From books, journals, govt reports
  • Examples: census data, published research
  • Quick to obtain but may be outdated
What is a frequency distribution? How is it formed?
A frequency distribution organizes raw data into classes/intervals and records the count (frequency) of values in each class. Steps: find range → choose class width → tally observations → count frequencies. Used to summarize and visualize large data sets.
What is a histogram? How is it different from a bar chart?
A histogram represents a frequency distribution of continuous data — bars are adjacent (no gaps), x-axis = class intervals, y-axis = frequency. A bar chart represents categorical/discrete data with gaps between bars. Histogram area = total frequency.

Measures of Central Tendency

🔥 HotWhat are mean, median, and mode? When is each preferred?
Mean = Σx / n (sum of all values ÷ count) Median = middle value when data is sorted Mode = most frequently occurring value
Use Mean when data is symmetric, no outliers. Use Median when data has outliers or is skewed (e.g. income). Use Mode for categorical data or to find the most common value.

Measures of Dispersion

🔥 HotWrite the formula for Standard Deviation. What does it measure?
Standard deviation σ measures the average spread of data around the mean.
Population SD: σ = √[ Σ(xᵢ − μ)² / N ] Sample SD: s = √[ Σ(xᵢ − x̄)² / (n−1) ]
Small σ → data is clustered near mean. Large σ → data is spread out.
What is the difference between Variance and Standard Deviation?
Variance = σ² (squared units, harder to interpret). Standard deviation = √Variance (same units as data, easier to interpret). Both measure spread, but SD is more commonly reported. Variance is used in mathematical derivations.
What is Covariance? How do you interpret its sign?
Covariance measures how two variables change together.
Cov(X,Y) = Σ(xᵢ − x̄)(yᵢ − ȳ) / n
Positive → both increase together. Negative → one increases as other decreases. Zero → no linear relationship. (Note: covariance is scale-dependent; use correlation for comparison.)

Skewness & Kurtosis

🔥 HotWhat is skewness? Explain positive and negative skewness.
Skewness measures the asymmetry of a distribution.
Positive Skew (Right)
  • Long tail on the right
  • Mean > Median > Mode
  • E.g. income distribution
Negative Skew (Left)
  • Long tail on the left
  • Mode > Median > Mean
  • E.g. age at retirement
Symmetric distribution: skewness = 0 (Normal).
What is Kurtosis? Explain leptokurtic, platykurtic, and mesokurtic.
Kurtosis measures the peakedness (tailedness) of a distribution relative to Normal.
Kurtosis = μ₄ / σ⁴ (4th standardized central moment)
Mesokurtic — Normal distribution, kurtosis = 3.
Leptokurtic — kurtosis > 3, sharper peak, heavier tails.
Platykurtic — kurtosis < 3, flatter peak, lighter tails.
IV

Correlation & Curve Fitting

Pearson's r · Rank Correlation · Regression · Least Squares · Curve Fitting

Correlation

🔥 HotWhat is correlation? What is the range of the correlation coefficient r?
Correlation measures the strength and direction of the linear relationship between two variables. The correlation coefficient r always lies in: −1 ≤ r ≤ +1.
r = +1 → perfect positive linear, r = −1 → perfect negative linear, r = 0 → no linear relationship.
🔥 HotWrite Pearson's formula for the correlation coefficient r.
r = Σ(xᵢ − x̄)(yᵢ − ȳ) / √[Σ(xᵢ−x̄)² · Σ(yᵢ−ȳ)²] Alternatively: r = [nΣxy − ΣxΣy] / √{[nΣx² − (Σx)²][nΣy² − (Σy)²]}
🔥 HotWhat is Spearman's Rank Correlation? When is it used? Give the formula.
Spearman's rank correlation is used when data is ordinal or when the relationship is monotonic but not linear. Ranks are assigned to each variable.
rₛ = 1 − [6ΣD²] / [n(n²−1)] where D = difference between ranks of each pair n = number of pairs
Range: −1 ≤ rₛ ≤ +1. Preferred over Pearson when data has outliers or is non-normal.
🔥 HotWhat is the difference between correlation and regression?
Correlation
  • Measures strength of relationship
  • Symmetric: r(X,Y) = r(Y,X)
  • Dimensionless (−1 to +1)
  • Does not predict values
Regression
  • Predicts value of one variable
  • Asymmetric: Y on X ≠ X on Y
  • Has units (depends on variables)
  • Gives equation of best-fit line

Regression Analysis

What is the regression line of Y on X? Write the equation.
The regression line of Y on X gives the best estimate of Y for a given X using the least squares method:
Y − Ȳ = b_yx · (X − X̄) where b_yx = r · (σ_y / σ_x) = Σ(x−x̄)(y−ȳ) / Σ(x−x̄)²
Minimizes the sum of squared vertical distances from data points to the line.

Curve Fitting by Least Squares

What is curve fitting? What is the Principle of Least Squares?
Curve fitting finds a mathematical function that best approximates a set of data points. The Principle of Least Squares states: choose the curve that minimizes the sum of squared residuals (differences between observed and fitted values):
Minimize: S = Σ(yᵢ − ŷᵢ)² = Σ[yᵢ − f(xᵢ)]²
★ FavWrite the normal equations for fitting a straight line y = a + bx.
Differentiating S w.r.t. a and b and setting to zero gives the normal equations:
Σy = na + b·Σx Σxy = a·Σx + b·Σx² Solve simultaneously for a (intercept) and b (slope).
How do you fit a power curve y = axᵇ using least squares?
Linearize by taking logarithm:
log y = log a + b·log x Let Y = log y, X = log x, A = log a → Y = A + bX (straight line form) Fit this line using normal equations, then a = antilog(A).
How do you fit an exponential curve y = aeᵇˣ?
Take natural log to linearize:
ln y = ln a + bx Let Y = ln y, A = ln a → Y = A + bx (straight line) Apply normal equations to (x, Y) data. Then a = eᴬ.
How do you fit a parabola y = a + bx + cx²?
The normal equations for a second-degree polynomial:
Σy = na + b·Σx + c·Σx² Σxy = a·Σx + b·Σx² + c·Σx³ Σx²y = a·Σx² + b·Σx³ + c·Σx⁴ Solve the 3×3 system for a, b, c.
