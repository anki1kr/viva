Skewness measure karna

🎯 Objective

💡 Concept

Skewness Ke Types

  • Skewness = 0 → Data perfectly symmetric (normal distribution)

⚡ Quick Revision — Sabhi Topics Ek Nazar Mein

Unit I — Random Variables & Distributions

Probability

Kisi event ke hone ka chance. P(A) = favourable / total. Range 0 to 1. Sum rule: P(A∪B) = P(A)+P(B)−P(A∩B). Product rule: P(A∩B) = P(A)·P(B|A).

Random Variable (RV)

Discrete RV — countable values (coin toss, dice). Continuous RV — range mein koi bhi value (height, weight). RV ek function hai jo outcomes ko numbers assign karta hai.

PMF / PDF / CDF

PMF (discrete) — P(X=x), sabka sum = 1. PDF (continuous) — f(x), area under curve = 1. CDF — F(x) = P(X≤x), 0 se 1 tak, non-decreasing. P(a<X≤b) = F(b)−F(a).

Discrete Distributions

Binomial — n trials, p success each. P(X=x) = ⁿCₓ·pˣ·(1−p)ⁿ⁻ˣ. Mean = np, Var = np(1−p).
Geometric — 1st success tak kitne trials. P(X=x) = (1−p)ˣ⁻¹·p. Memoryless.
Poisson — rare events, rate λ. P(X=x) = e⁻λ·λˣ/x!. Mean = Var = λ.

Continuous Distributions

Uniform — sab values equal likely in [a,b]. Mean = (a+b)/2.
Exponential — Poisson events ke beech ka wait time. Mean = 1/λ. Memoryless.
Normal — bell curve, μ aur σ se define hoti hai. 68–95–99.7 rule. Mean=Median=Mode. Z = (X−μ)/σ.

Unit II — Moments & MGF

Expectation (Mean)

E[X] = long-run average. Discrete: Σx·P(X=x). Continuous: ∫x·f(x)dx. Properties: E[aX+b] = a·E[X]+b. E[X+Y] = E[X]+E[Y] always. Var(X) = E[X²]−(E[X])².

Moments

Raw moment: μ'ᵣ = E[Xʳ]. Central moment: μᵣ = E[(X−μ)ʳ]. 1st central = 0, 2nd = Variance, 3rd relates to skewness, 4th to kurtosis.

MGF & PGF

MGF: M_X(t) = E[e^tX]. rth derivative at t=0 deta hai rth moment. Distribution uniquely identify karta hai. Independent ka MGF: M_{X+Y}(t) = M_X(t)·M_Y(t).
PGF: G_X(z) = E[z^X] — discrete RVs ke liye. G'(1) = E[X].

2D Random Variables — Joint / Marginal / Conditional

§1Introduction to Probability

Probability is a numerical measure of the likelihood of an event occurring. For a random experiment with equally likely outcomes:

P(A) = (favourable outcomes) / (total outcomes) 0 ≤ P(A) ≤ 1 P(sample space S) = 1, P(∅) = 0

Key terms:

  • Experiment — a process whose outcome is uncertain (e.g. tossing a coin).
  • Sample space (S) — set of all possible outcomes.
  • Event — any subset of S.
  • Mutually exclusive — events that cannot occur together (A ∩ B = ∅).
  • Independent — occurrence of one does not affect the other.

§2Probability Rules — Sum & Product, Joint Probability

(a) Sum Rule (Addition Theorem) — used when we want P(A or B).

General: P(A∪B) = P(A) + P(B) − P(A∩B) Mutually excl.: P(A∪B) = P(A) + P(B)

(b) Product Rule (Multiplication Theorem) — used when we want P(A and B).

General: P(A∩B) = P(A) · P(B|A) Independent: P(A∩B) = P(A) · P(B)

(c) Joint Probability P(A∩B) is the probability of A and B occurring together. For independent events it factors as the product of individual probabilities.

(d) Conditional Probability:

P(A|B) = P(A∩B) / P(B), provided P(B) > 0

§3Random Variables — Discrete & Continuous

A random variable (RV) is a function X : S → ℝ that assigns a real number to every outcome of a random experiment. We classify RVs by the kind of values they take.

  • Discrete RV — takes countable values (finite or countably infinite). E.g. number of heads in 5 tosses, number of defective items.
  • Continuous RV — takes any value in an interval of ℝ. E.g. height, weight, time, temperature.
Tip: If values can be listed (0,1,2,…) → discrete. If values fall in a range (e.g. 1.2 cm to 5.4 cm) → continuous.

§4Distribution Function (CDF) & Its Properties

The Cumulative Distribution Function (CDF) of a random variable X is:

F(x) = P(X ≤ x), x ∈ ℝ

Properties of F(x):

  • 0 ≤ F(x) ≤ 1 for all x.
  • F(−∞) = 0 and F(+∞) = 1.
  • F(x) is monotonically non-decreasing: if x₁ < x₂ then F(x₁) ≤ F(x₂).
  • F(x) is right-continuous.
  • P(a < X ≤ b) = F(b) − F(a).
  • For continuous RV: F'(x) = f(x), the PDF.

§5Probability Mass Function (PMF) & Probability Density Function (PDF)

PMF — for a discrete RV X, p(x) = P(X = x).

p(x) ≥ 0 for all x Σ p(x) = 1 (over all possible values)

PDF — for a continuous RV X, f(x) is the probability density.

f(x) ≥ 0 for all x ∫ f(x) dx = 1 (over (−∞, ∞)) P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx
Important: For continuous RV, P(X = c) = 0 for any single point c. Probability only makes sense over an interval.

§6Binomial Distribution

Models the number of successes in n independent Bernoulli trials, each with success probability p.

P(X = x) = ⁿCₓ · pˣ · qⁿ⁻ˣ, x = 0,1,2,…,n where q = 1 − p Mean = np Variance = npq SD = √(npq)

Conditions: fixed number of trials n, only two outcomes (success/failure), constant p, trials are independent.

Examples: number of heads in 10 tosses, number of defective bulbs in a batch.

§7Geometric Distribution

Models the number of trials until the first success in repeated independent Bernoulli trials.

P(X = x) = q^(x−1) · p, x = 1, 2, 3, … Mean = 1/p Variance = q / p²

Memoryless property: P(X > m + n | X > m) = P(X > n). The past doesn't influence future trials.

Example: number of dice throws till the first 6 appears.

§8Poisson Distribution

Models the number of rare events in a fixed interval of time/space when events occur independently at a constant average rate λ.

P(X = x) = e⁻λ · λˣ / x!, x = 0,1,2,… Mean = λ Variance = λ (always equal — characteristic property)

Conditions: n large, p small, np = λ finite. The Poisson is a limiting case of the Binomial when n → ∞ and p → 0 with np = λ.

Examples: number of typing mistakes per page, calls received at a switchboard per minute.

§9Uniform Distribution (Continuous)

All values in [a, b] are equally likely. Density is constant on the interval.

f(x) = 1/(b−a), a ≤ x ≤ b = 0 otherwise Mean = (a + b)/2 Variance = (b − a)² / 12

Use: random number generators, when nothing is known about the variable except its range.

§10Exponential Distribution

Models the waiting time between successive Poisson events.

f(x) = λ · e^(−λx), x ≥ 0 F(x) = 1 − e^(−λx) Mean = 1/λ Variance = 1/λ²

Memoryless property: P(X > s + t | X > s) = P(X > t).

Connection: if events follow Poisson(λ), the time between events is Exponential(λ).

Example: lifetime of an electronic component, time between two customers arriving.

§11Normal Distribution (Gaussian)

The most important continuous distribution. Bell-shaped, symmetric, completely described by mean μ and standard deviation σ.

f(x) = (1 / σ√(2π)) · e^(−(x−μ)² / 2σ²), −∞ < x < ∞ Mean = Median = Mode = μ Variance = σ²

Properties:

  • Symmetric about μ — skewness = 0.
  • Mesokurtic — kurtosis = 3.
  • Total area under curve = 1.
  • The 68–95–99.7 rule: about 68% of data lies in μ ± σ, 95% in μ ± 2σ, 99.7% in μ ± 3σ.
  • Standard Normal Z = (X − μ)/σ has mean 0 and SD 1; used with Z-tables.
Central Limit Theorem: the sum/mean of a large number of independent RVs approaches a Normal distribution, no matter the original distribution. This is why Normal appears everywhere.

§1Expectation of a Random Variable

The expected value E[X] is the long-run average value of X — its "centre of mass."

Discrete: E[X] = Σ x · P(X = x) Continuous: E[X] = ∫ x · f(x) dx General: E[g(X)] = Σ g(x) p(x) or ∫ g(x) f(x) dx

Properties of expectation:

  • E[c] = c for any constant c.
  • E[aX + b] = a·E[X] + b (linearity).
  • E[X + Y] = E[X] + E[Y] (always — even if dependent).
  • E[XY] = E[X]·E[Y] only if X and Y are independent.

Variance measures spread around the mean:

Var(X) = E[(X − μ)²] = E[X²] − (E[X])² SD(X) = √Var(X)

§2Moments — Raw & Central

Moments are expected values of powers of X and tell us about the shape of a distribution.

rth raw moment (about origin): μ'ᵣ = E[Xʳ] rth central moment (about mean): μᵣ = E[(X − μ)ʳ]

Interpretation:

  • μ'₁ = E[X] = mean.
  • μ₁ = 0 always.
  • μ₂ = Variance — measure of dispersion.
  • μ₃ — used in skewness (asymmetry).
  • μ₄ — used in kurtosis (peakedness).

§3Probability Generating Function (PGF) & Properties

Defined for non-negative integer-valued discrete RVs. Encodes the entire distribution as a power series.

G_X(z) = E[z^X] = Σ P(X = k) · zᵏ, k = 0,1,2,…

Properties:

  • G_X(1) = 1 always.
  • G'_X(1) = E[X].
  • G''_X(1) = E[X(X − 1)] → factorial moment.
  • Var(X) = G''_X(1) + G'_X(1) − [G'_X(1)]².
  • For X, Y independent: G_{X+Y}(z) = G_X(z) · G_Y(z).
  • PGF uniquely determines the distribution.

Examples: for Binomial(n,p): G(z) = (q + pz)ⁿ. For Poisson(λ): G(z) = e^{λ(z−1)}.

§4Moment Generating Function (MGF) & Properties

MGF generates all moments of an RV by differentiation.

M_X(t) = E[e^(tX)] Discrete: M_X(t) = Σ e^(tx) · p(x) Continuous: M_X(t) = ∫ e^(tx) · f(x) dx

How to extract moments:

E[Xʳ] = (d^r / dt^r) M_X(t) |_{t = 0} i.e. rth derivative at t = 0 gives the rth raw moment.

Properties:

  • M_X(0) = 1 always.
  • M_{aX+b}(t) = e^(bt) · M_X(at).
  • For X, Y independent: M_{X+Y}(t) = M_X(t) · M_Y(t).
  • Uniqueness: if two RVs have the same MGF (in a neighbourhood of 0), they have the same distribution.

Important MGFs:

  • Binomial(n,p): M(t) = (q + p·eᵗ)ⁿ
  • Poisson(λ): M(t) = exp[λ(eᵗ − 1)]
  • Normal(μ, σ²): M(t) = exp[μt + ½σ²t²]
  • Exponential(λ): M(t) = λ / (λ − t), t < λ

§5Two-Dimensional Random Variables — Joint Distribution

When two RVs X and Y are studied together, their joint distribution describes their combined behaviour.

Discrete: p(x,y) = P(X = x, Y = y) Σ Σ p(x,y) = 1 Continuous: f(x,y) ≥ 0, ∬ f(x,y) dx dy = 1 P((X,Y) ∈ A) = ∬_A f(x,y) dx dy

The joint CDF is F(x,y) = P(X ≤ x, Y ≤ y).

§6Marginal Distributions

The marginal distribution of one variable is obtained by summing/integrating the joint distribution over the other.

Discrete: f_X(x) = Σ_y p(x,y) f_Y(y) = Σ_x p(x,y) Continuous: f_X(x) = ∫ f(x,y) dy f_Y(y) = ∫ f(x,y) dx

Each marginal is itself a valid 1D distribution (sums/integrates to 1).

§7Conditional Distributions

The conditional distribution of X given Y = y describes how X is distributed when Y is fixed.

f(x | y) = f(x, y) / f_Y(y), provided f_Y(y) > 0 f(y | x) = f(x, y) / f_X(x), provided f_X(x) > 0

Conditional expectation:

E[X | Y = y] = Σ x · f(x|y) or ∫ x · f(x|y) dx

§8Independence of Random Variables

Two RVs X and Y are independent iff their joint distribution factorises into the product of their marginals.

f(x, y) = f_X(x) · f_Y(y) for all x, y Equivalently: f(x | y) = f_X(x) F(x, y) = F_X(x) · F_Y(y)

Consequences of independence:

  • E[XY] = E[X] · E[Y].
  • Cov(X, Y) = 0.
  • Var(X + Y) = Var(X) + Var(Y).
  • M_{X+Y}(t) = M_X(t) · M_Y(t).
Warning: Cov(X, Y) = 0 does not imply independence (it only rules out linear association).

§1Types of Data — Primary & Secondary

Primary Data — collected first-hand by the investigator for a specific purpose. Original, fresh and reliable.

  • Methods: direct observation, interviews, questionnaires, experiments, surveys.
  • Advantage: accurate, purpose-specific.
  • Limitation: time-consuming and costly.

Secondary Data — already collected by some other agency / individual, reused for the present study.

  • Sources: published reports, journals, govt publications, census, websites.
  • Advantage: quick and inexpensive.
  • Limitation: may be outdated or not exactly relevant.

Other classifications:

  • Qualitative (categorical) vs Quantitative (numeric).
  • Quantitative further: Discrete (counts) vs Continuous (measurements).

§2Classification & Representation of Data

Classification arranges raw data into homogeneous groups for easier analysis. Bases of classification:

  • Geographical — by location (state-wise, city-wise).
  • Chronological — by time (year-wise, month-wise).
  • Qualitative — by attributes (gender, religion).
  • Quantitative — by numerical magnitude (income, marks).

Representation of data:

  • Tabular — frequency tables, contingency tables.
  • Diagrammatic — bar diagrams, pie charts, pictograms.
  • Graphical — histogram, frequency polygon, frequency curve, ogive (cumulative frequency curve).

§3Formation of Frequency Distribution

A frequency distribution groups raw data into class intervals with the count (frequency) of observations in each class.

Steps:

  • Find Range = (Max − Min).
  • Decide number of classes (typically 5–15).
  • Class width h ≈ Range / Number of classes.
  • Form non-overlapping classes (e.g. 0–10, 10–20, …).
  • Tally each observation into its class.
  • Count tally marks → frequency f.

Types:

  • Discrete frequency distribution — exact values with frequencies.
  • Continuous (grouped) — class intervals with frequencies.
  • Cumulative frequency — running total (less-than or more-than).
  • Relative frequency — f / N (proportion).

§4Measures of Central Tendency — Mean, Median, Mode

Central tendency summarises data with a single representative value.

(a) Arithmetic Mean (x̄)

Raw data: x̄ = Σx / n Frequency data: x̄ = Σf·x / Σf Grouped data: x̄ = Σf·m / Σf (m = mid-value of class)

(b) Median — middle value of sorted data; not affected by outliers.

Ungrouped (n odd): median = ((n+1)/2)th value Ungrouped (n even): median = avg of (n/2)th and (n/2 + 1)th Grouped: median = L + [ (N/2 − F) / f ] · h L = lower limit of median class, F = cf before, f = freq, h = width

(c) Mode — value(s) with highest frequency.

Grouped: mode = L + [ (f₁ − f₀) / (2f₁ − f₀ − f₂) ] · h f₁ = freq of modal class, f₀ = previous, f₂ = next

Empirical relation (moderately skewed):

Mode = 3·Median − 2·Mean

§5Standard Deviation & Variance

Standard Deviation (σ) is the most important measure of dispersion — the RMS deviation from the mean.

Population: σ = √[ Σ(xᵢ − μ)² / N ] Sample: s = √[ Σ(xᵢ − x̄)² / (n − 1) ] For frequency data: σ = √[ Σf(x − x̄)² / Σf ]

Variance = σ² (units squared).

Properties of SD:

  • SD ≥ 0, with σ = 0 only when all values are equal.
  • SD is unaffected by change of origin (shift), but is scaled by change of scale.
  • Lower SD → data clustered around mean; higher SD → spread out.
  • Coefficient of Variation CV = (σ / x̄) × 100 % (dimensionless, useful to compare two data sets).

§6Covariance

Covariance measures the joint variability of two variables X and Y — how they change together.

Cov(X, Y) = Σ(xᵢ − x̄)(yᵢ − ȳ) / n = E[XY] − E[X] · E[Y]

Interpretation:

  • Cov > 0 → X and Y tend to move in the same direction.
  • Cov < 0 → X and Y move in opposite directions.
  • Cov = 0 → no linear relationship (independence ⇒ Cov = 0, but not vice versa).
Limitation: covariance is scale-dependent — its magnitude changes with units. Use the correlation coefficient r = Cov(X,Y) / (σ_x · σ_y) for a unit-free measure.

§7Skewness — Asymmetry of a Distribution

Skewness measures the lack of symmetry of a distribution about its mean.

Karl Pearson's coeff. (Sk): Sk = (Mean − Mode) / SD = 3(Mean − Median) / SD (when mode is unstable) Bowley's coeff. (based on quartiles): Sk_B = (Q₃ + Q₁ − 2·Median) / (Q₃ − Q₁) Moment-based: β₁ = μ₃² / μ₂³, γ₁ = μ₃ / σ³

Types:

  • Positive skew (right-skewed) — long tail on right; Mean > Median > Mode (e.g. income).
  • Negative skew (left-skewed) — long tail on left; Mode > Median > Mean (e.g. retirement age).
  • Symmetric (skewness = 0) — Mean = Median = Mode (Normal distribution).

§8Kurtosis — Peakedness of a Distribution

Kurtosis measures the relative peakedness (and tail heaviness) of a distribution compared to the Normal.

β₂ = μ₄ / μ₂² = μ₄ / σ⁴ γ₂ = β₂ − 3 (excess kurtosis)

Types:

  • Mesokurtic — β₂ = 3, γ₂ = 0. The Normal distribution.
  • Leptokurtic — β₂ > 3. Tall, narrow peak with heavy tails (more outliers).
  • Platykurtic — β₂ < 3. Flat, broad peak with thin tails.
Memory tip: Lepto = leaping high (tall peak), Platy = flat plate (low peak), Meso = middle (normal).

What is a frequency distribution? How is it formed?
A frequency distribution organizes raw data into classes/intervals and records the count (frequency) of values in each class. Steps: find range → choose class width → tally observations → count frequencies. Used to summarize and visualize large data sets.
What is a histogram? How is it different from a bar chart?
A histogram represents a frequency distribution of continuous data — bars are adjacent (no gaps), x-axis = class intervals, y-axis = frequency. A bar chart represents categorical/discrete data with gaps between bars. Histogram area = total frequency.

Measures of Central Tendency

🔥 HotWhat are mean, median, and mode? When is each preferred?
Mean = Σx / n (sum of all values ÷ count) Median = middle value when data is sorted Mode = most frequently occurring value
Use Mean when data is symmetric, no outliers. Use Median when data has outliers or is skewed (e.g. income). Use Mode for categorical data or to find the most common value.

Measures of Dispersion

🔥 HotWrite the formula for Standard Deviation. What does it measure?
Standard deviation σ measures the average spread of data around the mean.
Population SD: σ = √[ Σ(xᵢ − μ)² / N ] Sample SD: s = √[ Σ(xᵢ − x̄)² / (n−1) ]
Small σ → data is clustered near mean. Large σ → data is spread out.
What is the difference between Variance and Standard Deviation?
Variance = σ² (squared units, harder to interpret). Standard deviation = √Variance (same units as data, easier to interpret). Both measure spread, but SD is more commonly reported. Variance is used in mathematical derivations.
What is Covariance? How do you interpret its sign?
Covariance measures how two variables change together.
Cov(X,Y) = Σ(xᵢ − x̄)(yᵢ − ȳ) / n
Positive → both increase together. Negative → one increases as other decreases. Zero → no linear relationship. (Note: covariance is scale-dependent; use correlation for comparison.)

Skewness & Kurtosis

🔥 HotWhat is skewness? Explain positive and negative skewness.
Skewness measures the asymmetry of a distribution.
Positive Skew (Right)
  • Long tail on the right
  • Mean > Median > Mode
  • E.g. income distribution
Negative Skew (Left)
  • Long tail on the left
  • Mode > Median > Mean
  • E.g. age at retirement
Symmetric distribution: skewness = 0 (Normal).
What is Kurtosis? Explain leptokurtic, platykurtic, and mesokurtic.
Kurtosis measures the peakedness (tailedness) of a distribution relative to Normal.
Kurtosis = μ₄ / σ⁴ (4th standardized central moment)
Mesokurtic — Normal distribution, kurtosis = 3.
Leptokurtic — kurtosis > 3, sharper peak, heavier tails.
Platykurtic — kurtosis < 3, flatter peak, lighter tails.

§1Correlation Analysis

Correlation measures the degree and direction of linear relationship between two variables X and Y.

Types of correlation:

  • Positive — X and Y move together (height & weight).
  • Negative — X increases, Y decreases (price & demand).
  • No (zero) correlation — no linear pattern.
  • Linear — points roughly on a straight line.
  • Non-linear (curvilinear) — pattern follows a curve.

Methods: scatter diagram, Karl Pearson's coefficient, Spearman's rank correlation, concurrent deviation method.

§2Karl Pearson's Coefficient of Correlation (r)

Measures the strength of linear relationship between X and Y.

r = Σ(x − x̄)(y − ȳ) / √[ Σ(x − x̄)² · Σ(y − ȳ)² ] Alternate computational form: r = [ nΣxy − Σx·Σy ] / √{ [nΣx² − (Σx)²] · [nΣy² − (Σy)²] } Also: r = Cov(X, Y) / (σ_x · σ_y)

Properties:

  • Range: −1 ≤ r ≤ +1.
  • r = +1 → perfect positive correlation; r = −1 → perfect negative.
  • r = 0 → no linear relationship (could still have non-linear).
  • r is symmetric: r(X, Y) = r(Y, X).
  • r is unit-free and unaffected by change of origin and scale.
  • |r| ≥ 0.7 → strong; 0.3–0.7 → moderate; < 0.3 → weak.

§3Spearman's Rank Correlation

Used when data is ordinal, has outliers, or the relationship is monotonic but not linear. Each observation is replaced by its rank.

rₛ = 1 − [ 6·ΣD² ] / [ n(n² − 1) ] D = difference between ranks of corresponding x and y n = number of pairs

When ranks are tied — assign the average of the ranks that would have been given, and apply the correction factor:

rₛ = 1 − [ 6·{ ΣD² + Σ(m³ − m)/12 } ] / [ n(n² − 1) ] m = number of items sharing a tied rank in each tie

Range: −1 ≤ rₛ ≤ +1. Interpretation same as Pearson's r.

§4Regression Analysis

Regression establishes a mathematical relationship to predict the value of one variable from another. Unlike correlation, it is asymmetric: the regression of Y on X is different from X on Y.

Two regression lines:

Y on X (predict Y given X): Y − ȳ = b_yx (X − x̄) b_yx = r · σ_y / σ_x = Σ(x − x̄)(y − ȳ) / Σ(x − x̄)² X on Y (predict X given Y): X − x̄ = b_xy (Y − ȳ) b_xy = r · σ_x / σ_y = Σ(x − x̄)(y − ȳ) / Σ(y − ȳ)²

Properties of regression coefficients:

  • b_yx · b_xy = r² → r = ±√(b_yx · b_xy). Sign of r matches sign of the coefficients.
  • Both regression coefficients have the same sign.
  • |b_yx · b_xy| ≤ 1 (since r² ≤ 1).
  • Both regression lines pass through the point (x̄, ȳ).
  • If r = 0, the two lines are perpendicular; if |r| = 1, they coincide.

§5Curve Fitting & Principle of Least Squares

Curve fitting determines the best mathematical function y = f(x) that approximates a set of observed (xᵢ, yᵢ) data.

Principle of Least Squares: choose the curve that minimises the sum of squared residuals between observed and fitted values.

Minimise: S = Σ (yᵢ − ŷᵢ)² = Σ [ yᵢ − f(xᵢ) ]² Take ∂S/∂(parameter) = 0 for each parameter → "normal equations".
Why squares? Squaring (i) removes signs so positive & negative errors don't cancel, (ii) penalises large errors more, (iii) yields linear normal equations with a unique solution.

§6Fitting a Straight Line y = a + bx

Differentiating S = Σ(y − a − bx)² with respect to a and b and equating to zero gives the two normal equations:

Σy = n·a + b·Σx Σxy = a·Σx + b·Σx² Solve simultaneously for: b (slope) = [ nΣxy − Σx·Σy ] / [ nΣx² − (Σx)² ] a (intercept) = ȳ − b·x̄

The line y = a + bx is the line of best fit, passing through (x̄, ȳ).

§7Fitting a Parabola (Second-Degree) y = a + bx + cx²

Three parameters a, b, c need three normal equations (obtained by partial derivatives of S):

Σy = n·a + b·Σx + c·Σx² Σxy = a·Σx + b·Σx² + c·Σx³ Σx²y = a·Σx² + b·Σx³ + c·Σx⁴ Solve the 3×3 system for a, b, c.

Used when scatter shows a parabolic (curved) pattern.

§8Fitting a Power Curve y = a·xᵇ

The curve is non-linear in x but becomes linear in log–log scale. Linearise first:

Take log of both sides: log y = log a + b · log x Substitute Y = log y, X = log x, A = log a: Y = A + b·X (straight-line form) Apply normal equations to (X, Y) data: ΣY = n·A + b·ΣX ΣXY = A·ΣX + b·ΣX² Finally: a = antilog(A).

§9Fitting an Exponential Curve y = a·eᵇˣ

Linearise using natural logarithm:

Take ln of both sides: ln y = ln a + b·x Substitute Y = ln y, A = ln a: Y = A + b·x (straight-line form) Normal equations: ΣY = n·A + b·Σx ΣxY = A·Σx + b·Σx² Solve and recover a = eᴬ.

Note: only the solution for parameters changes — the principle (least squares) is the same as fitting a straight line.

Exam tip: as per syllabus — no derivation, numerical problems only. Memorise the normal equations and the linearisation substitutions; then it's just arithmetic.

🔥 HotWhat is the difference between correlation and regression?
Correlation
  • Measures strength of relationship
  • Symmetric: r(X,Y) = r(Y,X)
  • Dimensionless (−1 to +1)
  • Does not predict values
Regression
  • Predicts value of one variable
  • Asymmetric: Y on X ≠ X on Y
  • Has units (depends on variables)
  • Gives equation of best-fit line

Regression Analysis

What is the regression line of Y on X? Write the equation.
The regression line of Y on X gives the best estimate of Y for a given X using the least squares method:
Y − Ȳ = b_yx · (X − X̄) where b_yx = r · (σ_y / σ_x) = Σ(x−x̄)(y−ȳ) / Σ(x−x̄)²
Minimizes the sum of squared vertical distances from data points to the line.
