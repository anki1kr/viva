1.1 Probability Laws — Addition always 1.5M

Addition Law If A and B are two events, then
P(A or B) = P(A ∪ B) = P(A) + P(B) − P(A ∩ B) If A & B are mutually exclusive events: P(A ∩ B) = 0 ⇒ P(A ∪ B) = P(A) + P(B)

Q · 53 Sundays 1.5 M

What is the probability of getting 53 Sundays in (i) a leap year, (ii) a non-leap year?

i · leap yearNo. of days = 366 = 52 weeks + 2 days.

2 days may be: {Sun-Mon, Mon-Tue, Tue-Wed, Wed-Thu, Thu-Fri, Fri-Sat, Sat-Sun} → total cases = 7, favourable = 2 (Sun-Mon & Sat-Sun).

P(53 Sundays | leap) = 2/7

ii · non-leapNo. of days = 365 = 52 weeks + 1 day.

1 day may be: {Sun, Mon, Tue, Wed, Thu, Fri, Sat} → total = 7, favourable = 1.

P(53 Sundays | non-leap) = 1/7

Q · Lottery tickets 1.5 M

20 lottery tickets numbered 1–20 are mixed in a box and 1 ticket is drawn at random. Find the probability that the drawn number is a multiple of 3 or 5.

step 1Sample space S = {1, 2, …, 20}, n(S) = 20.

step 2Let A = multiples of 3 = {3, 6, 9, 12, 15, 18}, n(A) = 6.

Let B = multiples of 5 = {5, 10, 15, 20}, n(B) = 4.

A ∩ B = {15}, n(A ∩ B) = 1.

step 3P(A ∪ B) = 6/20 + 4/20 − 1/20 = 9/20.

P(multiple of 3 or 5) = 9/20 = 0.45

1.2 Conditional Probability

Definition If A and B are two events, the conditional probability of A when B has already happened is
P(A | B) = P(A ∩ B) / P(B) , where P(B) ≠ 0 Similarly: P(B | A) = P(A ∩ B) / P(A) , where P(A) ≠ 0
Independent events If A and B are independent:
P(A ∩ B) = P(A) · P(B)

1.3 Multiplication Rule

Multiplication law
P(A and B) = P(A ∩ B) = P(A) · P(B | A) P(A ∩ B ∩ C) = P(A) · P(B | A) · P(C | A∩B) If A, B, C are independent: P(A ∩ B ∩ C) = P(A) · P(B) · P(C)

Q · Die thrown twice — 4 appears at least once 7 M

A die is thrown twice and the sum of the numbers is observed to be 6. What is the conditional probability that the number 4 has appeared at least once?

step 1Total outcomes = 36.

step 2A = "4 appears at least once" = { (1,4), (2,4), (3,4), (4,1), (4,2), (4,3), (4,4), (4,5), (4,6), (5,4), (6,4) } → n(A) = 11.

step 3B = "sum is 6" = { (1,5), (2,4), (3,3), (4,2), (5,1) } → n(B) = 5 ⇒ P(B) = 5/36.

step 4A ∩ B = {(2,4), (4,2)} → n = 2 ⇒ P(A ∩ B) = 2/36.

step 5P(A | B) = P(A ∩ B) / P(B) = (2/36) / (5/36) = 2/5.

P(4 at least once | sum = 6) = 2/5

Q · Three students — problem solved 7 M

Probability of solving a maths problem by three students A, B, C is 1/2, 1/3, 1/4 respectively. What is the probability that the problem is solved?

step 1P(A) = 1/2 ⇒ P(Ā) = 1 − 1/2 = 1/2.

P(B) = 1/3 ⇒ P(B̄) = 2/3.

P(C) = 1/4 ⇒ P(C̄) = 3/4.

step 2P(not solved) = P(Ā ∩ B̄ ∩ C̄) = P(Ā)·P(B̄)·P(C̄) (independent) = (1/2)(2/3)(3/4) = 6/24 = 1/4.

step 3P(solved) + P(not solved) = 1.

P(solved) = 1 − 1/4 = 3/4.

P(problem is solved) = 3/4

1.4 Total Probability & Bayes' Theorem 8M heavy

Total Probability If E₁, E₂, …, Eₙ are mutually exclusive and exhaustive events, and A is any event associated with Eᵢ:
P(A) = Σᵢ₌₁ⁿ P(Eᵢ) · P(A | Eᵢ) = P(E₁)·P(A|E₁) + P(E₂)·P(A|E₂) + … + P(Eₙ)·P(A|Eₙ)
Bayes' Theorem For the same partition E₁, E₂, …, Eₙ and event A:
P(Eᵢ | A) = P(Eᵢ) · P(A | Eᵢ) / Σⱼ₌₁ⁿ P(Eⱼ) · P(A | Eⱼ) For 3 events, the denominator becomes: P(E₁)·P(A|E₁) + P(E₂)·P(A|E₂) + P(E₃)·P(A|E₃)

Q · Three bags Bayes 8 M

Three bags contain (3 red, 4 green) ; (4 red, 5 green) ; (5 red, 6 green) balls respectively. A bag is selected at random and one ball is drawn which is found to be red. What is the probability that the drawn ball came from the second bag?

step 1P(E₁) = P(E₂) = P(E₃) = 1/3.

step 2Let A = event of red ball drawn.

P(A | E₁) = 3/7 (3 red out of 7).

P(A | E₂) = 4/9 (4 red out of 9).

P(A | E₃) = 5/11 (5 red out of 11).

step 3By Bayes' Theorem,

P(E₂ | A) = P(E₂)·P(A|E₂) / [P(E₁)·P(A|E₁) + P(E₂)·P(A|E₂) + P(E₃)·P(A|E₃)] = (1/3)·(4/9) / [(1/3)(3/7) + (1/3)(4/9) + (1/3)(5/11)] = (4/27) ÷ (1/3)·[3/7 + 4/9 + 5/11] = (4/9) ÷ [3/7 + 4/9 + 5/11]

step 4LCM(7, 9, 11) = 693.

3/7 = 297/693, 4/9 = 308/693, 5/11 = 315/693.

Sum = (297 + 308 + 315) / 693 = 920/693.

step 5P(E₂ | A) = (4/9) ÷ (920/693) = (4 × 693) / (9 × 920) = 2772/8280 = 77/230 ≈ 0.29.

P(from 2nd bag | red) = 77/230 ≈ 0.29

1.5 Random Variable

Definition A random variable is a function which assigns each outcome of a sample space S to a real number. It is denoted by X.

Example: If a coin is tossed twice and X = number of heads, then values of X are 0, 1, 2.

Two types
  • Discrete RV — X takes countable values only (e.g. X = 0, 1, 2, …).
  • Continuous RV — X takes all possible values within a range.

Probability Distribution For a probability distribution, the sum of all probabilities is equal to 1:
Σ Pᵢ = 1

Discrete Probability Distribution If a random variable X takes values x₁, x₂, …, xₙ with corresponding probabilities P₁, P₂, …, Pₙ such that Σ Pᵢ = 1, then occurrence of xᵢ with pᵢ is called a discrete probability distribution.
Probability Mass Function (PMF) If X is a discrete RV taking values x₁, x₂, … with probabilities P₁, P₂, …, then PMF is defined as
P(x) = P(X = xᵢ) = Pᵢ

Conditions:

  • P(xᵢ) ≥ 0
  • Σ Pᵢ = 1
Cumulative Distribution Function (CDF)
F(x) = P(X ≤ xᵢ)

Example: If P(X = 1) = 2/4, then

CDF F(1) = P(X ≤ 1) = P(X = 0) + P(X = 1) = 1/4 + 2/4 = 3/4.

1.7 Mean & Variance of Discrete RV

Mean / Expectation If X takes values x₁, x₂, …, xₙ with probabilities P₁, P₂, …, Pₙ, then expectation (mean) of X is
μ = E(X) = Σᵢ₌₁ⁿ xᵢ · Pᵢ E(X²) = Σᵢ₌₁ⁿ xᵢ² · Pᵢ
Variance
Var(X) = E(X²) − [E(X)]²
Properties of E(X) For a constant a (and another RV Y):
  • E(a) = a
  • E(aX + b) = a · E(X) + b
  • E(X · Y) = E(X) · E(Y)  (if X, Y independent)
  • E(aX + bY) = a · E(X) + b · E(Y)

1.8 Binomial Distribution 8M certain

Binomial PMF A random variable X is said to have a binomial distribution if its PMF is
P(X = x) = ⁿCₓ · pˣ · q^(n−x) where: p = probability of success q = probability of failure (p + q = 1) n = number of trials x = number of successes ⁿCₓ = n! / [x! · (n − x)!]
Mean & Variance
Mean of Binomial with parameters n, p: E(X) = μ = n·p Variance: Var(X) = σ² = n·p·q

Q · Coin tossed 10 times — at least 7 heads 8 M

A coin is tossed 10 times. What is the probability of getting at least 7 heads?

step 1p = P(head) = 1/2, q = 1/2, n = 10.

step 2P(at least 7 heads) = P(X = 7) + P(X = 8) + P(X = 9) + P(X = 10).

= ¹⁰C₇·(1/2)⁷·(1/2)³ + ¹⁰C₈·(1/2)⁸·(1/2)² + ¹⁰C₉·(1/2)⁹·(1/2)¹ + ¹⁰C₁₀·(1/2)¹⁰·(1/2)⁰ Use ⁿCᵣ = ⁿC_(n−r): ¹⁰C₇ = ¹⁰C₃ = (10·9·8)/(3·2·1) = 120 ¹⁰C₈ = ¹⁰C₂ = (10·9)/(2·1) = 45 ¹⁰C₉ = ¹⁰C₁ = 10 ¹⁰C₁₀ = 1

step 3Each term has (1/2)¹⁰ as common factor:

= (1/2)¹⁰ · (120 + 45 + 10 + 1) = (1/2)¹⁰ · 176 = 176 / 1024 = 11 / 64 ≈ 0.172

P(at least 7 heads) = 11/64 ≈ 0.17

1.9 Poisson Distribution 8M favourite

Definition Poisson is the limiting case of the Binomial distribution as n → ∞ (large) and p → 0 (small).
P(X = x) = e^(−λ) · λˣ / x! where λ = n·p is called the parameter. Mean = Variance = λ

Suppose a book of 585 pages contains 43 typographical errors. If the errors are randomly distributed throughout the book, what is the probability that 10 pages selected at random have no error? (Given e^(−0.735) = 0.4795.)

1.10 Continuous Random Variable

Definition If a random variable X takes all possible values in a given range, then it is called a continuous random variable.
Probability Density Function (PDF) PDF of a continuous random variable X is given by
f(x) = P(a ≤ x ≤ b) = ∫ₐᵇ f(x) dx
where f(x) satisfies:
  • f(x) ≥ 0
  • ∫₋∞^(+∞) f(x) dx = 1
CDF, Mean, Variance
CDF: F(x) = P(X ≤ x) = ∫₋∞ˣ f(x) dx Mean: E(X) = ∫₋∞^∞ x · f(x) dx E(X²) = ∫₋∞^∞ x² · f(x) dx Var: Var(X) = E(X²) − [E(X)]²

Q · f(x) = Kx³ on [0, 3] 8 M

If the PDF is f(x) = Kx³ for 0 ≤ x ≤ 3 and 0 otherwise, find (i) value of K, (ii) P(1 ≤ X ≤ 2), (iii) Mean & Variance.

∫₋∞⁰ f(x)dx + ∫₀³ Kx³ dx + ∫₃^∞ f(x)dx = 1 0 + ∫₀³ Kx³ dx + 0 = 1 ∫₀³ Kx³ dx = K · [x⁴/4]₀³ = K · (81/4 − 0) = 81K/4 ⇒ 81K/4 = 1 ⇒ K = 4/81

K = 4/81

ii · P(1 ≤ X ≤ 2)

= ∫₁² Kx³ dx = K · [x⁴/4]₁² = K · (16 − 1)/4 = 15K/4 = (15/4) × (4/81) = 15/81 = 5/27

P(1 ≤ X ≤ 2) = 5/27

iii · MeanE(X) = ∫₀³ x · Kx³ dx = K · ∫₀³ x⁴ dx

= K · [x⁵/5]₀³ = K · 243/5 = (4/81) × (243/5) = (4 × 3)/5 = 12/5

iii · E(X²)= ∫₀³ x² · Kx³ dx = K · ∫₀³ x⁵ dx

1.11 Normal Distribution 8M near-certain

Definition A continuous RV X is said to have a normal distribution if its PDF is
f(x) = 1 / (σ√(2π)) · e^(−½ · ((x − μ)/σ)²) −∞ < x < ∞, σ > 0 where μ & σ are the parameters of the distribution μ = mean σ = standard deviation
Properties
  • Mean = Median = Mode (all coincide)
  • Curve is symmetrical about mean and bell-shaped
  • Total area under the normal probability curve = 1
  • Area under μ − σ < x < μ + σ = 68 %
  • Area under μ − 2σ < x < μ + 2σ = 95 %
  • Area under μ − 3σ < x < μ + 3σ = 99 %
Standard Normal Variate
Z = (X − μ) / σ Symmetry rule: φ(−Z) = 1 − φ(Z) Range: P(a ≤ Z ≤ b) = φ(b) − φ(a)

Q · 100 dry battery cells — Normal 8 M

A sample of 100 dry battery cells gives μ = 12 hours, σ = 3 hours. Assuming the data is normally distributed, what % of cells are expected to have life: (i) more than 15 hours, (ii) less than 6 hours, (iii) between 10 and 14 hours?

Let X denote the length of life of battery.

i · X > 15Z = (15 − 12)/3 = 3/3 = 1.

P(X > 15) = P(Z > 1) = 1 − P(Z ≤ 1) = 1 − 0.8413 = 0.1587 = 15.87 %

P(life > 15 h) = 15.87 %

ii · X < 6Z = (6 − 12)/3 = −6/3 = −2.

P(X < 6) = P(Z < −2) = P(Z > 2) (by symmetry) = 1 − P(Z ≤ 2) = 1 − φ(2) = 1 − 0.9772 = 0.0228 = 2.28 %

P(life < 6 h) = 2.28 %

iii · 10 ≤ X ≤ 14

When x = 10: Z = (10 − 12)/3 = −0.67 When x = 14: Z = (14 − 12)/3 = +0.67 P(10 ≤ X ≤ 14) = P(−0.67 ≤ Z ≤ 0.67) = 2 · P(0 ≤ Z ≤ 0.67) = 2 · [φ(0.67) − φ(0)] = 2 · [0.7486 − 0.5000] = 2 · 0.2486 = 0.4960 = 49.60 %

P(10 ≤ life ≤ 14) = 49.60 %

1.12 Exponential Distribution

Definition A continuous random variable X has the exponential distribution if its PDF is
f(x) = λ · e^(−λx), x ≥ 0 0 , otherwise where λ is the parameter. Mean: E(X) = 1/λ = λ⁻¹ Variance: Var(X) = 1/λ²
Useful identities
  • P(X > t) = e^(−λt)
  • P(X ≤ t) = 1 − e^(−λt)
  • Memoryless property — P(X > s + t | X > s) = P(X > t)

2.1 Mathematical Expectation — E[X] always asked

Definition The expected value (mean) of a random variable X is
E[X] = Σ x · P(X = x) (discrete) E[X] = ∫₋∞^∞ x · f(x) dx (continuous)
Properties of E[X]
  • E[c] = c
  • E[aX + b] = a·E[X] + b
  • E[X + Y] = E[X] + E[Y] (always)
  • E[X · Y] = E[X] · E[Y] only if X, Y independent
  • E[X²] = Σ x² · P(X = x)

Q · Find E[X] and E[2X+3] 4 M

P(X=1)=0.2, P(X=2)=0.5, P(X=3)=0.3. Find E[X] and E[2X+3].

E[X]= 1×0.2 + 2×0.5 + 3×0.3 = 0.2 + 1.0 + 0.9 = 2.1.

E[2X+3]= 2·E[X] + 3 = 2×2.1 + 3 = 4.2 + 3.

E[X] = 2.1  ·  E[2X+3] = 7.2

2.2 Variance & Standard Deviation

Variance Formula
Var(X) = E[X²] − (E[X])² Standard Deviation: σ = √Var(X)
Properties of Variance
  • Var(c) = 0
  • Var(aX + b) = a²·Var(X)
  • Var(X + Y) = Var(X) + Var(Y) if X, Y independent

Q · Find Var(X) 5 M

X takes values 1, 2, 3 with P = 1/6, 1/2, 1/3. Find Var(X).

E[X]= 1·(1/6) + 2·(1/2) + 3·(1/3) = 1/6 + 3/6 + 6/6 · ½ ... = 13/6.

E[X²]= 1·(1/6) + 4·(1/2) + 9·(1/3) = 1/6 + 12/6 + 18/6 = 31/6.

Var= 31/6 − (13/6)² = 186/36 − 169/36.

Var(X) = 17/36 ≈ 0.472

2.3 Moment Generating Function (MGF)

Definition
M_X(t) = E[e^(tX)] Discrete: M_X(t) = Σ e^(tx) · P(X = x) Continuous: M_X(t) = ∫ e^(tx) · f(x) dx r-th moment: E[Xʳ] = M_X^(r)(0)
Viva Tip MGF uniquely determines a distribution. If two RVs have same MGF → same distribution.

2.4 Chebyshev's Inequality definition Q

Statement For any RV X with mean μ and variance σ², for any k > 0:
P(|X − μ| ≥ k·σ) ≤ 1/k² Equivalently: P(|X − μ| < k·σ) ≥ 1 − 1/k² k=2 → at least 75% within 2σ k=3 → at least 88.9% within 3σ

Q · Chebyshev bound 3 M

X has mean 50 and SD 10. Find minimum P(30 < X < 70).

Range = μ ± 2σ, so k = 2.

P(|X − 50| < 20) ≥ 1 − 1/4.

Minimum P = 3/4 = 0.75

2.5 Binomial — Mean & Variance always asked

Conditions + Formulas
  • Fixed n trials, each gives success (p) or failure (q = 1−p)
  • Trials are independent, p is constant
P(X = x) = ⁿCₓ · pˣ · q^(n−x) Mean: E[X] = np Variance: Var(X) = npq SD: σ = √(npq)

Q · Find n, p, q given mean and variance 5 M

For a binomial distribution, mean = 4 and variance = 3. Find n, p, q.

step 1np = 4, npq = 3.

step 2npq/np = q = 3/4 → p = 1/4.

step 3n × (1/4) = 4 → n = 16.

n = 16 · p = 1/4 · q = 3/4

2.6 Poisson — Mean = Variance = λ trick Q

When to use Poisson n is very large, p is very small, λ = np is finite.
P(X = x) = e^(−λ) · λˣ / x! Mean = Variance = λ

Q · Bank customers — P(none) and P(≥1) 5 M

4.2 Karl Pearson's Correlation Coefficient 8M numerical

Formula

  • Symmetric: r(X,Y) = r(Y,X)
  • Unaffected by change of origin or scale
  • −1 ≤ r ≤ 1 always

4.4 Regression — Two Lines lines of best fit

Two Regression Lines
Line of Y on X (to predict Y): y − ȳ = byx · (x − x̄) byx = r · (σy / σx) Line of X on Y (to predict X): x − x̄ = bxy · (y − ȳ) bxy = r · (σx / σy) Key relation: r² = byx · bxy so r = ±√(byx · bxy)
When do both lines coincide? When r = ±1 (perfect correlation). Both lines become one when all points lie on a straight line.
Least Squares Principle Regression line minimises the sum of squared vertical distances (residuals) from the line.
Normal equations for Y = a + bX: Σy = na + b·Σx Σxy = a·Σx + b·Σx²

Q 1(b) · Conditions for Poisson 1.5 M

Conditions under which Poisson distribution is used.

Poisson(λ) models the number of rare events in a fixed interval. Conditions:

  1. Number of trials n is very large
  2. Probability of success p is very small
  3. Product np = λ is finite (moderate)
  4. Events occur independently at constant average rate
  5. Two events cannot occur at the same instant

Examples: typing errors per page · phone calls per minute · defective items per lot.

Q 1(c) · Verify pdf 1.5 M

Prove that f(x) = 6x(1 − x), 0 ≤ x ≤ 1, is a probability density function.

Q 1(e) · Properties of joint distribution function 1.5 M

Write down the properties of joint distribution function.

For two RVs X and Y, joint CDF is defined as

F(x, y) = P(X ≤ x, Y ≤ y)

Properties:

  1. 0 ≤ F(x, y) ≤ 1
  2. F(−∞, y) = F(x, −∞) = 0
  3. F(+∞, +∞) = 1
  4. F is non-decreasing & right-continuous in each variable
  5. Marginals: F_X(x) = F(x, +∞), F_Y(y) = F(+∞, y)
  6. P(a < X ≤ b, c < Y ≤ d) = F(b, d) − F(a, d) − F(b, c) + F(a, c)

Q 1(f) · Define covariance 1.5 M

Define covariance. What is the covariance of two independent variables?

Definition.

Cov(X, Y) = E[(X − μ_X)(Y − μ_Y)] = E(XY) − E(X)·E(Y)

Covariance measures the direction of linear association.

  • Cov > 0 → X, Y move together
  • Cov < 0 → opposite direction
  • Cov = 0 → no linear association

For independent X, Y: E(XY) = E(X)·E(Y), so

Q 7(b) · Two dice — sum divisible by 2 or 3 7 M

Two dice are thrown together. Find the probability that the sum is divisible by 2 or 3.

Formula (addition rule with overlap):

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Total outcomes = 6 × 6 = 36. Possible sums: 2 to 12.

step 1 · ASum divisible by 2 (even): {2, 4, 6, 8, 10, 12}.

Counts: 1 + 3 + 5 + 5 + 3 + 1 = 18 ⇒ P(A) = 18/36 = 1/2.

step 2 · BSum divisible by 3: {3, 6, 9, 12}.

Counts: 2 + 5 + 4 + 1 = 12 ⇒ P(B) = 12/36 = 1/3.

step 3 · A ∩ BSum divisible by both 2 and 3, i.e. by 6: {6, 12}.
