01 Probability Foundations UNIT I

Probability measures the chance that an event happens. It always lies between 0 (impossible) and 1 (certain). Three definitions coexist: classical (equally likely outcomes), empirical (long-run frequency), and axiomatic (Kolmogorov's three rules).

1.1 Basic Vocabulary

TermMeaningExample
ExperimentA process whose outcome is uncertainTossing a coin
Sample space (S)Set of all possible outcomesS = {H, T}
Event (E)A subset of SE = {H} ("heads")
Mutually exclusiveCannot occur together (E₁ ∩ E₂ = ∅)"Heads" & "Tails"

Classical: P(E) = number of favourable outcomes / total outcomes Empirical: P(E) = frequency of E in n trials / n Axiomatic: 0 ≤ P(E) ≤ 1, P(S) = 1, P(A∪B) = P(A)+P(B) if disjoint

1.2 Sum Rule (Addition)

Definition For any two events A and B:
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
If A and B are mutually exclusive, then P(A ∩ B) = 0, so P(A ∪ B) = P(A) + P(B).
Sum Rule — inclusion-exclusion
Counting A and B separately double-counts A ∩ B — subtract it once.
Example A card is drawn from a 52-card deck. P(King) = 4/52, P(Heart) = 13/52, P(King of Hearts) = 1/52.
P(King OR Heart) = 4/52 + 13/52 − 1/52 = 16/52 = 4/13.

1.3 Product Rule (Multiplication)

Definition
P(A ∩ B) = P(A) · P(B | A) If independent: P(A ∩ B) = P(A) · P(B)

1.4 Joint, Marginal & Conditional Probability

Joint

P(A ∩ B) — both A and B happen.

Marginal

P(A) — A alone, ignoring B. Sum the joint over all B values.

Conditional

P(A | B) — A given that B has happened.

P(A|B) = P(A∩B) / P(B)

Probability Tree — toss two coins
Multiply along branches (product rule); add across leaves (sum rule).
Memory hook "AND multiplies, OR adds (minus the overlap)"
PYQ — May 2025 · Q2(b)7 marks Two dice are thrown together. Find the probability that the sum of numbers of points on them is divisible by 2 or 3.
Solution. Sample space size = 6 × 6 = 36.

Let A = "sum divisible by 2" and B = "sum divisible by 3".

Sums divisible by 2 (i.e. 2, 4, 6, 8, 10, 12):
frequencies = 1 + 3 + 5 + 5 + 3 + 1 = 18 ⇒ P(A) = 18/36

Sums divisible by 3 (i.e. 3, 6, 9, 12):
frequencies = 2 + 5 + 4 + 1 = 12 ⇒ P(B) = 12/36

Sums divisible by both (i.e. divisible by 6 → 6, 12):
frequencies = 5 + 1 = 6 ⇒ P(A ∩ B) = 6/36

P(A ∪ B) = P(A) + P(B) − P(A ∩ B) = 18/36 + 12/36 − 6/36 = 24/36 = 2/3 ≈ 0.667.

Quick Check — Topic 1

Q1. State the sum rule for two non-disjoint events.
P(A ∪ B) = P(A) + P(B) − P(A ∩ B). The overlap is subtracted once.
Q2. When is P(A ∩ B) = P(A) · P(B)?
When A and B are independent — the occurrence of one does not affect the probability of the other.
Q3. Two coins are tossed. P(at least one head)?
1 − P(no heads) = 1 − ½·½ = ¾.

02 Random Variables & Distribution Functions UNIT I

A random variable (RV) is a function that assigns a real number to every outcome in a sample space. It turns "messy" outcomes into numbers we can compute on. RVs split into discrete (countable values) and continuous (uncountable, e.g. real numbers).

Random variable maps outcomes to numbers

2.1 Probability Mass Function (PMF) — discrete RVs

Definition For a discrete RV X, the PMF is p(x) = P(X = x) — the probability that X takes the specific value x.
Properties: 1. p(x) ≥ 0 2. Σ p(x) = 1 (sum over all values of X)

2.2 Probability Density Function (PDF) — continuous RVs

Definition For a continuous RV X, the PDF f(x) is a non-negative function such that the probability of X falling into an interval is the area under f.
Properties: 1. f(x) ≥ 0 2. ∫ f(x) dx = 1 (over entire real line) 3. P(a ≤ X ≤ b) = ∫ₐᵇ f(x) dx 4. P(X = c) = 0 for any single c ← key difference from PMF

2.3 Cumulative Distribution Function (CDF)

Definition The CDF is F(x) = P(X ≤ x) — the probability that X is at most x. Defined for both discrete and continuous RVs.
Properties: 1. F(−∞) = 0, F(+∞) = 1 2. Monotone non-decreasing: if a ≤ b ⇒ F(a) ≤ F(b) 3. Right-continuous 4. Discrete: F(x) = Σ p(xᵢ) for xᵢ ≤ x (step function) Continuous: F(x) = ∫₋∞ˣ f(t) dt (smooth) 5. P(a < X ≤ b) = F(b) − F(a)
PMF (discrete) vs PDF (continuous) vs CDF

2.4 Expectation & Variance

Mean (Expectation): Discrete: E[X] = μ = Σ x · p(x) Continuous: E[X] = μ = ∫ x · f(x) dx Variance: Var(X) = σ² = E[(X − μ)²] = E[X²] − μ² Standard deviation: σ = √Var(X) Linearity (key!): E[X + Y] = E[X] + E[Y] (always — no independence needed)
PYQ — May 2025 · Q1(a)1.5 marks Define distribution function for a random variable.
The distribution function (or cumulative distribution function, CDF) of a random variable X is
F(x) = P(X ≤ x), defined for every real x.
It satisfies: (1) F(−∞) = 0 and F(+∞) = 1; (2) it is monotone non-decreasing; (3) it is right-continuous.
For discrete X, F is a step function: F(x) = Σ p(xᵢ) for xᵢ ≤ x.
For continuous X, F(x) = ∫₋∞ˣ f(t) dt and f(x) = dF/dx.
PYQ — May 2025 · Q1(c)1.5 marks Prove that f(x) = 6x(1 − x) for 0 ≤ x ≤ 1 is a probability density function.
Two conditions to verify.

(i) Non-negativity: for 0 ≤ x ≤ 1 both x ≥ 0 and (1 − x) ≥ 0, so 6x(1 − x) ≥ 0. ✓

(ii) Total area = 1:
∫₀¹ 6x(1 − x) dx = 6 ∫₀¹ (x − x²) dx
= 6 [x²/2 − x³/3]₀¹
= 6 [1/2 − 1/3] = 6 · (1/6) = 1. ✓

Both conditions satisfied ⇒ f(x) is a valid PDF.
PYQ — May 2025 · Q3(b)7 marks Two unbiased dice are thrown. Find the expectation of the sum of points on them.
Solution. Let X = points on die 1, Y = points on die 2. Each is uniform on {1,2,3,4,5,6}.

E[X] = (1 + 2 + 3 + 4 + 5 + 6)/6 = 21/6 = 3.5
E[Y] = 3.5 (same)

By linearity of expectation:
E[X + Y] = E[X] + E[Y] = 3.5 + 3.5 = 7.

Verification by direct computation:
E[X+Y] = Σ s · P(sum = s) for s = 2..12
= (2·1 + 3·2 + 4·3 + 5·4 + 6·5 + 7·6 + 8·5 + 9·4 + 10·3 + 11·2 + 12·1)/36
= 252/36 = 7 ✓

Quick Check — Topic 2

Q1. What's the difference between a PMF and a PDF?
PMF gives P(X = x) directly for discrete RVs (a probability). PDF gives the density f(x) for continuous RVs — probability is the integral over an interval, and P(X = c) = 0 for any single value.
Q2. List two properties every CDF must satisfy.
Any two of: F(−∞)=0; F(+∞)=1; monotone non-decreasing; right-continuous; P(a<X≤b)=F(b)−F(a).
Q3. Variance shortcut formula?
Var(X) = E[X²] − (E[X])² — easier than computing E[(X−μ)²] directly.

03 Discrete Distributions UNIT I

Three discrete distributions cover the bulk of exam questions: Binomial (count of successes in n trials), Geometric (trials until first success), and Poisson (count of rare events in fixed interval).

3.1 Binomial Distribution

Setup Fixed number of trials n, each independent, each with success probability p. Let X = number of successes. Then X ~ B(n, p).
PMF: P(X = k) = C(n, k) · pᵏ · (1−p)ⁿ⁻ᵏ , k = 0, 1, …, n Mean: μ = np Var: σ² = np(1−p) SD: σ = √(np(1−p))
Binomial PMF — B(10, 0.5)
Example A coin is tossed 10 times. P(exactly 7 heads) = C(10,7) · 0.5⁷ · 0.5³ = 120 · (1/1024) = 0.117.

3.2 Geometric Distribution

Setup Independent Bernoulli trials with success probability p. Let X = number of trials needed to get the first success. Then X ~ Geom(p).
PMF: P(X = k) = (1−p)ᵏ⁻¹ · p , k = 1, 2, 3, … Mean: μ = 1/p Var: σ² = (1−p)/p² Memoryless: P(X > m+n | X > m) = P(X > n)
Example A die is rolled until a 6 appears. p = 1/6, so expected rolls = 1/p = 6.

3.3 Poisson Distribution

Setup Count of rare events in a fixed time interval (or area) when events occur independently at constant average rate λ. Then X ~ Poisson(λ).
PMF: P(X = k) = e⁻λ · λᵏ / k! , k = 0, 1, 2, … Mean: μ = λ Var: σ² = λ ← unique: mean = variance!
Tip — Poisson approximates Binomial When n is large and p is small (n ≥ 20, p ≤ 0.05), B(n, p) ≈ Poisson(λ = np). Useful when n! is huge.
Example A call centre averages λ = 4 calls/min. P(exactly 2 calls in next minute) = e⁻⁴ · 4²/2! = 0.01832 · 16/2 = 0.1465.

3.4 Side-by-Side Summary

Distribution"Counts…"MeanVariance

Memory hook "Binomial counts, Geometric waits, Poisson sprays rare events"
PYQ — May 2025 · Q1(b)1.5 marks What are the conditions under which Poisson distribution is used?
Poisson(λ) is appropriate when:
(1) Events occur independently of each other.
(2) Events occur in a fixed interval (time, area, volume) at a constant average rate λ.
(3) The probability of an event in a very small interval is proportional to its length, and the probability of two events in that small interval is negligible.
(4) n is large and p is small (rule of thumb: n ≥ 20, p ≤ 0.05) so that B(n, p) ≈ Poisson(np).
(5) Counts are non-negative integers (0, 1, 2, …) with no fixed upper bound.
PYQ — May 2025 · Q2(a)8 marks In a lot of 200 articles, 10 are defective. Find the probability of (i) no defective article, (ii) one defective article, (iii) at least one defective article in a random sample of 20 articles.
Solution (Binomial model). p = 10/200 = 0.05, q = 1 − p = 0.95, n = 20.
X = number of defective articles in the sample, X ~ B(20, 0.05).

(i) P(X = 0) = C(20,0) · (0.05)⁰ · (0.95)²⁰ = (0.95)²⁰ ≈ 0.3585

(ii) P(X = 1) = C(20,1) · (0.05)¹ · (0.95)¹⁹ = 20 · 0.05 · (0.95)¹⁹
= 1 · (0.95)¹⁹ = (0.95)¹⁹ ≈ 0.3774

(iii) P(X ≥ 1) = 1 − P(X = 0) = 1 − 0.3585 = 0.6415

Cross-check via Poisson approx (λ = np = 1):
P(X=0) ≈ e⁻¹ = 0.3679, P(X=1) ≈ e⁻¹ = 0.3679, P(X≥1) ≈ 0.6321 — close to Binomial values.

Quick Check — Topic 3

Q1. Mean & variance of B(20, 0.3).
μ = np = 6, σ² = np(1−p) = 20·0.3·0.7 = 4.2.
Q2. What's special about the Poisson distribution's parameters?
Mean = Variance = λ — the only common distribution where these are equal.
Q3. State the memoryless property of the Geometric distribution.
P(X > m + n | X > m) = P(X > n) — past failures don't change the future trial-count distribution.

04 Continuous Distributions UNIT I

Three continuous distributions you must master: Uniform (equal density on an interval), Exponential (waiting time between rare events), and Normal (the bell curve — most-asked in any exam).

4.1 Uniform Distribution U(a, b)

PDF: f(x) = 1 / (b − a) , for a ≤ x ≤ b , else 0 Mean: μ = (a + b) / 2 Var: σ² = (b − a)² / 12 CDF: F(x) = (x − a) / (b − a)
Uniform PDF — flat on [a, b]

4.2 Exponential Distribution

Setup Waiting time until the next Poisson event (e.g. time until next call). Parameter λ = rate.
PDF: f(x) = λ · e⁻λˣ , x ≥ 0 (λ > 0) Mean: μ = 1 / λ Var: σ² = 1 / λ² CDF: F(x) = 1 − e⁻λˣ Memoryless: P(X > s+t | X > s) = P(X > t)
Exponential PDF — λ controls steepness

4.3 Normal Distribution N(μ, σ²)

Setup The famous "bell curve". By the Central Limit Theorem, sums/averages of many independent RVs tend to be normal — which is why it's everywhere.
PDF: f(x) = (1 / (σ√(2π))) · exp( −(x − μ)² / (2σ²) ) Mean: μ SD: σ Var: σ² Standard Normal: Z = (X − μ) / σ , Z ~ N(0, 1)
Normal Distribution — 68-95-99.7 rule
68% within 1σ, 95% within 2σ, 99.7% within 3σ — the "empirical rule".

4.4 Standard Normal & Z-scores

Worked example Heights are N(170, 10²). What % of people are taller than 185 cm?
Z = (185 − 170) / 10 = 1.5.
From Z-table: P(Z > 1.5) ≈ 0.0668 → about 6.7%.

PYQ — May 2025 · Q3(a)8 marks A sample of 100 dry-battery cells gave Mean μ = 12 hours, SD σ = 3 hours. Assuming normality, what percentage of cells are expected to have life (i) more than 15 hours, (ii) less than 6 hours, (iii) between 10 and 14 hours?
Solution. Standardise: Z = (X − 12)/3.

(i) P(X > 15): Z = (15 − 12)/3 = 1.0
P(Z > 1) = 0.5 − P(0 < Z < 1) = 0.5 − 0.3413 = 0.1587 → 15.87%
So about 15.87 ≈ 16 cells last more than 15 hours.

(ii) P(X < 6): Z = (6 − 12)/3 = −2.0
P(Z < −2) = 0.5 − P(0 < Z < 2) = 0.5 − 0.4772 = 0.0228 → 2.28%
So about 2 cells last less than 6 hours.

(iii) P(10 < X < 14): Z₁ = (10 − 12)/3 = −0.67, Z₂ = (14 − 12)/3 = 0.67
P(−0.67 < Z < 0.67) = 2 · P(0 < Z < 0.67) = 2 · 0.2486 = 0.4972 → 49.72%
So about 50 cells last between 10 and 14 hours.

Quick Check — Topic 4

Q1. Mean of U(2, 10)?
(a + b)/2 = (2 + 10)/2 = 6.
Q2. State the empirical 68-95-99.7 rule.
In a Normal distribution, ~68% of values lie within 1σ of the mean, ~95% within 2σ, ~99.7% within 3σ.
Q3. Z-score formula?
Z = (X − μ) / σ — measures how many SDs X is from the mean.
Q4. For X ~ N(50, 10²), find P(X > 60).
Z = (60 − 50)/10 = 1. P(Z > 1) = 0.1587 ≈ 15.87%.

PYQ — May 2025 · Q5(a)8 marks Define data and its types. How do we classify and represent it? Explain with suitable examples.
Data = facts, figures, observations or measurements collected for analysis. It is the raw material of statistics.

Types of data:
(a) By nature — Qualitative (attribute, e.g. gender, religion) vs Quantitative (numerical, e.g. height, marks). Quantitative further splits into discrete (countable: number of students) and continuous (measurable: weight).

06 Measures of Central Tendency UNIT III

A single number that describes the "centre" of a dataset. Three popular measures: Mean (average), Median (middle), and Mode (most frequent).

6.1 Mean (Arithmetic Average)

Ungrouped: x̄ = Σ xᵢ / n Grouped: x̄ = Σ fᵢ · xᵢ / Σ fᵢ where xᵢ is the class midpoint, fᵢ the frequency Weighted: x̄_w = Σ wᵢ xᵢ / Σ wᵢ
Example Marks: 4, 6, 8, 10, 12 → mean = (4+6+8+10+12)/5 = 40/5 = 8.

6.2 Median (Middle Value)

Step 1: Sort the data in ascending order. Step 2: if n is odd: median = the ((n+1)/2)th value if n is even: median = average of the two middle values Grouped data: Median = L + ((n/2 − cf) / f) · h L = lower limit of the median class cf = cumulative frequency before the median class f = frequency of the median class h = class width
Example Sorted: 3, 7, 9, 11, 15. n=5 (odd). Median = 3rd value = 9.
If data were 3, 7, 9, 11, 15, 18 (n=6), median = (9+11)/2 = 10.

6.3 Mode (Most Frequent Value)

Ungrouped: the value that appears most often Grouped data: Mode = L + ((f₁ − f₀) / (2f₁ − f₀ − f₂)) · h L = lower limit of the modal class f₁ = frequency of the modal class f₀ = frequency of the class before f₂ = frequency of the class after h = class width
Mean · Median · Mode on a distribution

6.4 Combined / Pooled Mean (multi-group)

For two groups: x̄₁₂ = (n₁·x̄₁ + n₂·x̄₂) / (n₁ + n₂) For k groups: x̄ = Σ nᵢ x̄ᵢ / Σ nᵢ

6.5 Empirical Relation

For a moderately skewed distribution: Mode ≈ 3 · Median − 2 · Mean

6.6 When to use which?

MeasureBest when…Affected by outliers?
MeanData is symmetric, no extreme outliersYes — pulled toward outliers
MedianSkewed data, has outliers (e.g. income)No — robust
ModeCategorical data; spotting most commonNo
Memory hook "Mean adds, Median locates, Mode counts"
PYQ — May 2025 · Q1(g)1.5 marks The consumption of printing paper reams (in units) for the first 11 months is 20, 25, 30, 15, 17, 35, 26, 18, 40, 45, 50. Find the median.
Solution.
Step 1 — sort: 15, 17, 18, 20, 25, 26, 30, 35, 40, 45, 50.
Step 2 — n = 11 (odd), so median = the ((n+1)/2)th = 6th observation.
Median = 26 reams.
PYQ — May 2025 · Q5(b)7 marks A cooperative bank has two branches employing 50 and 70 workers respectively. The average salary paid by the two branches is Rs. 360 and Rs. 390 per month. Calculate the mean salary of all employees.
Solution. Use the combined-mean formula.
n₁ = 50, x̄₁ = 360;   n₂ = 70, x̄₂ = 390.

x̄ = (n₁ x̄₁ + n₂ x̄₂) / (n₁ + n₂)
= (50 × 360 + 70 × 390) / (50 + 70)
= (18,000 + 27,300) / 120
= 45,300 / 120
= Rs. 377.50 per month.

Sanity check: the answer (377.50) lies between 360 and 390 and is closer to 390 because the larger branch (70 workers) has the higher mean. ✓

Quick Check — Topic 6

Sorted (already), n=5 (odd) → middle = 11.
Q2. Why is median preferred for income data?
Income is right-skewed with extreme high earners. The mean is pulled up by them, but the median (middle value) is robust to outliers.

07 Dispersion & Covariance UNIT III

Central tendency tells you the centre. Dispersion tells you how spread out the values are around it. Covariance measures how two variables move together.

7.1 Range

Range = Max value − Min value

Simplest measure but very sensitive to outliers.

7.2 Variance & Standard Deviation

Population variance: σ² = Σ (xᵢ − μ)² / N Sample variance: s² = Σ (xᵢ − x̄)² / (n − 1) ← Bessel's correction Standard deviation: σ = √σ² Computational form: σ² = (Σ xᵢ²)/N − μ² Grouped data: σ² = (Σ fᵢ(xᵢ − x̄)²) / Σ fᵢ
Tip — why divide by (n−1) for samples? Using n for samples underestimates the true population variance. Dividing by n−1 ("Bessel's correction") gives an unbiased estimate.
Standard deviation visualises spread
Same mean, different spread — the bigger the σ, the wider and flatter the distribution.

7.3 Coefficient of Variation

CV = (σ / μ) × 100%

A unit-free comparison of dispersion across datasets with different means.

7.4 Covariance — two variables together

Definition Covariance measures the joint variability of two random variables X and Y. Positive ⇒ they tend to move together; negative ⇒ they move opposite; zero ⇒ no linear relation.
Population: Cov(X, Y) = (1/N) · Σ (xᵢ − μ_X)(yᵢ − μ_Y) Sample: Cov(X, Y) = (1/(n−1)) · Σ (xᵢ − x̄)(yᵢ − ȳ) Sign meaning: Cov > 0 → X and Y rise together Cov < 0 → one rises while the other falls Cov = 0 → no linear relationship
Covariance — three patterns
PYQ — May 2025 · Q1(f)1.5 marks Define covariance. What is the covariance of two independent variables?
Definition. Covariance of X and Y is
Cov(X, Y) = E[(X − μ_X)(Y − μ_Y)] = E[XY] − E[X] · E[Y].
It measures the joint variability/linear association of X and Y.

If X and Y are independent, then E[XY] = E[X] · E[Y], so
Cov(X, Y) = 0.

Caution: the converse is not true — Cov = 0 does not imply independence (e.g., Y = X² with X symmetric about 0).

Quick Check — Topic 7

Q1. Why divide sample variance by (n−1) instead of n?
Bessel's correction — dividing by n underestimates the population variance; (n−1) gives an unbiased estimator.
Q2. What does Cov(X, Y) < 0 mean?
X and Y move in opposite directions — when one increases, the other tends to decrease.
Q3. If X and Y are independent, what is Cov(X, Y)? Is the converse true?
Cov(X, Y) = 0. The converse is false — zero covariance does not imply independence (only no linear relationship).
Q4. State the formula for sample covariance.
Cov(X, Y) = (1/(n − 1)) · Σ (xᵢ − x̄)(yᵢ − ȳ).

08 Skewness UNIT III

Skewness measures the asymmetry of a distribution. A perfectly symmetric distribution has skewness = 0. Real-world data is rarely symmetric — incomes are right-skewed, exam marks of an easy paper are left-skewed.

Three shapes of skewness

8.1 Identifying Skewness

ShapeTail extendsMean vs Median vs ModeReal-world example
SymmetricBoth sides equalMean = Median = ModeHeights of adults
Positive (right) skewTo the rightMode < Median < MeanIncome, house prices
Negative (left) skewTo the leftMean < Median < ModeAge at death, easy exam scores

8.2 Karl Pearson's Coefficient of Skewness

Sk_p = (Mean − Mode) / SD When mode is hard to find (multimodal): Sk_p = 3 · (Mean − Median) / SD Range: −3 ≤ Sk_p ≤ +3 (typically between −1 and +1)

8.3 Bowley's Coefficient (uses quartiles)

Sk_b = (Q₃ + Q₁ − 2 · Q₂) / (Q₃ − Q₁) Q₁ = first quartile, Q₂ = median, Q₃ = third quartile Range: −1 ≤ Sk_b ≤ +1

8.4 Interpreting Skewness Values

Sk ≈ 0Symmetric distribution
Sk > 0Right-skewed (positive)
Sk < 0Left-skewed (negative)
|Sk| > 1Highly skewed
0.5 < |Sk| < 1Moderately skewed
|Sk| < 0.5Approximately symmetric
Worked example A class scored: Mean = 60, Median = 65, Mode = 70, SD = 10.
Sk_p = (60 − 70)/10 = −1.0 → moderately negatively skewed (long left tail; most students did well, a few did poorly).
PYQ — May 2025 · Q6(a)8 marks Compute the coefficient of skewness from the figures: 25, 15, 23, 40, 27, 25, 23, 25, 20.
Step 1 — sort & find centre.

Median = 5th observation = 25.

Step 2 — standard deviation. Compute (xᵢ − x̄)²:

(25−24.78)² = 0.05, (25−24.78)² = 0.05, (25−24.78)² = 0.05, (27−24.78)² = 4.93,
(40−24.78)² = 231.65.

Memory hook "The tail tells the tale — skew sign matches the tail direction"
Right tail → positive skew. Left tail → negative skew. Mean follows the tail.

Q1 · Define probability and state its three approaches Probability of an event E is a numerical measure of the chance of its occurrence, satisfying 0 ≤ P(E) ≤ 1.
(i) Classical: P(E) = favourable outcomes / total outcomes (assumes equally likely).
(ii) Empirical / Relative frequency: P(E) = limn→∞ (f / n).
(iii) Axiomatic (Kolmogorov): P(E) ≥ 0, P(S) = 1, and for disjoint events P(A∪B) = P(A) + P(B).
Example: P(even number on a fair die) = 3/6 = 0.5.
Q2 · State and prove conditional probability formula Definition: P(A|B) = P(A ∩ B) / P(B), provided P(B) > 0.
Derivation: Restrict the sample space from S to B. The favourable outcomes for A inside B are exactly A ∩ B. Hence the proportion is P(A ∩ B)/P(B).
Example: Card drawn from 52. P(King | Face card) = P(King ∩ Face)/P(Face) = (4/52)/(12/52) = 1/3.
Q3 · Bayes' Theorem statement & tiny example For events A₁, A₂, …, Aₙ partitioning S and any event B with P(B) > 0:
P(Aᵢ | B) = P(B | Aᵢ) · P(Aᵢ) / Σⱼ P(B | Aⱼ) · P(Aⱼ)
Example: Disease prevalence 1%, test 99% sensitive, 95% specific. P(disease | +) = (0.99)(0.01) / [(0.99)(0.01) + (0.05)(0.99)] = 0.0099 / 0.0594 ≈ 0.167.
Q4 · Random variable + PMF vs PDF A random variable is a real-valued function X : S → ℝ that maps each outcome to a number.
PMF (discrete): p(x) = P(X = x), with Σ p(xᵢ) = 1.
PDF (continuous): f(x) ≥ 0 with ∫−∞∞ f(x) dx = 1, and P(a ≤ X ≤ b) = ∫ab f(x) dx.
Example: Toss two coins, X = number of heads. PMF: p(0)=¼, p(1)=½, p(2)=¼. Σ = 1 ✓.
Q5 · Expectation and variance — short derivation Expectation: E(X) = Σ xᵢ · p(xᵢ) (discrete) or ∫ x · f(x) dx (continuous).
Variance: Var(X) = E[(X − μ)²] = E(X²) − μ². Standard deviation σ = √Var(X).
Example: Die roll. E(X) = (1+2+3+4+5+6)/6 = 3.5. E(X²) = (1+4+9+16+25+36)/6 = 91/6 ≈ 15.17. Var(X) = 15.17 − 12.25 = 2.92.
Q6 · Bernoulli vs Binomial Bernoulli(p): single trial with success probability p. P(X=1)=p, P(X=0)=1−p. Mean = p, Variance = pq.
Binomial(n, p): n independent Bernoulli trials. P(X=k) = C(n,k) pᵏ qⁿ⁻ᵏ. Mean = np, Variance = npq.
Example: Toss a fair coin 4 times. P(exactly 2 heads) = C(4,2)·(½)²·(½)² = 6 · 1/16 = 6/16 = 0.375.
Q7 · Poisson distribution — when & how Models the number of rare events in a fixed interval. P(X = k) = e−λ · λᵏ / k!, k = 0, 1, 2, …
Mean = Variance = λ. Used when n is large and p is small with np = λ moderate (Binomial → Poisson approximation).
Example: A help-desk receives λ = 2 calls/min. P(exactly 3 calls in a minute) = e⁻² · 2³ / 3! = (0.1353)(8)/6 ≈ 0.180.
Q8 · Normal distribution properties PDF: f(x) = (1 / σ√(2π)) · exp[−(x − μ)² / 2σ²].
(i) Bell-shaped and symmetric about x = μ. (ii) Mean = Median = Mode = μ. (iii) ≈68 % of area lies within μ ± σ, ≈95 % within μ ± 2σ, ≈99.7 % within μ ± 3σ. (iv) Total area = 1.
Example: If X ~ N(50, 100), then σ = 10. P(40 ≤ X ≤ 60) = P(−1 ≤ Z ≤ 1) ≈ 0.6826.
Q9 · Mean, Median, Mode — definitions and use Mean: x̄ = Σxᵢ / n — best for symmetric data.
Median: middle value of ordered data — robust to outliers.
Mode: most frequent value — works for categorical data.
Example: 4, 7, 7, 9, 13. Mean = 40/5 = 8, Median = 7, Mode = 7.
Q10 · Range, Variance, Standard Deviation Range = max − min (very rough; uses only extremes).
Variance: σ² = Σ(xᵢ − x̄)² / n.
SD: σ = √σ² (same units as data).
Example: {2, 4, 4, 4, 5, 5, 7, 9}. x̄ = 5. Σ(xᵢ−5)² = 9+1+1+1+0+0+4+16 = 32. σ² = 32/8 = 4 ⇒ σ = 2.
Q11 · Coefficient of variation CV = (SD / Mean) × 100 % — a unit-free measure that lets you compare spread across data sets with different scales.
Example: Set A: mean = 50, σ = 5 ⇒ CV = 10 %. Set B: mean = 200, σ = 10 ⇒ CV = 5 %. Although Set B has larger SD, Set A is more variable relative to its mean.
Q12 · Karl Pearson coefficient of skewness Sk_p = (Mean − Mode) / SD. If mode is undefined or unstable, use Sk = 3·(Mean − Median)/SD.
Sk > 0 ⇒ right-skewed, Sk < 0 ⇒ left-skewed, Sk = 0 ⇒ symmetric. Typically |Sk| ≤ 1.
Example: Mean 60, Mode 50, SD 10 ⇒ Sk = (60 − 50)/10 = +1.0 (highly right-skewed).

Q1 · State and prove Bayes' theorem. Solve a numerical example.8 marks Statement. Let A₁, A₂, …, Aₙ be a partition of the sample space S (mutually exclusive and exhaustive, all P(Aᵢ) > 0). For any event B with P(B) > 0,
P(Aᵢ | B) = P(B | Aᵢ) · P(Aᵢ) / Σⱼ₌₁ⁿ P(B | Aⱼ) · P(Aⱼ)
Proof. By definition, P(Aᵢ | B) = P(Aᵢ ∩ B) / P(B). Using the multiplication rule, P(Aᵢ ∩ B) = P(B | Aᵢ) · P(Aᵢ). By the total probability theorem, P(B) = Σⱼ P(B | Aⱼ) · P(Aⱼ). Substituting both gives the stated result. ∎ Numerical. Three machines M₁, M₂, M₃ produce 50 %, 30 %, 20 % of items with defective rates 1 %, 2 %, 3 % respectively. An item is found defective; what is the probability it came from M₂?

Step 1 — priors and likelihoods.
P(M₁) = 0.5, P(M₂) = 0.3, P(M₃) = 0.2.
P(D | M₁) = 0.01, P(D | M₂) = 0.02, P(D | M₃) = 0.03.
Step 2 — total P(D).
P(D) = (0.5)(0.01) + (0.3)(0.02) + (0.2)(0.03) = 0.005 + 0.006 + 0.006 = 0.017.
Step 3 — apply Bayes.
P(M₂ | D) = (0.02 × 0.3) / 0.017 = 0.006 / 0.017 ≈ 0.353.

Use case: spam filtering, medical diagnosis, machine fault attribution — anywhere a posterior must be updated from prior + new evidence.

Q3 · Define the Poisson distribution. Derive mean and variance and solve an example.8 marks Definition. X ~ Poisson(λ) if
P(X = k) = e^(−λ) · λᵏ / k!, k = 0, 1, 2, …
Models the number of independent rare events in a fixed interval of time / space when n → ∞, p → 0 with np = λ. Mean. E(X) = Σ k · e⁻λ λᵏ/k! = λ · e⁻λ Σ λ^(k−1)/(k−1)! = λ · e⁻λ · e^λ = λ.
Variance. E[X(X−1)] = λ², so Var(X) = E(X²) − μ² = (λ² + λ) − λ² = λ.
Result: Mean = Variance = λ — the signature property of Poisson. When to use: n large, p small, np = λ moderate (typically n > 30 and p < 0.05). Worked example. A book has on average λ = 1.5 typographical errors per page. Find the probability that a randomly chosen page contains (a) no errors, (b) exactly 2 errors, (c) at most 1 error.
(a) P(X = 0) = e⁻¹·⁵ · 1 = 0.2231.
(b) P(X = 2) = e⁻¹·⁵ · (1.5)² / 2! = 0.2231 · 2.25 / 2 = 0.2231 · 1.125 ≈ 0.2510.
(c) P(X ≤ 1) = P(0) + P(1) = 0.2231 + 0.2231·1.5 = 0.2231 + 0.3347 ≈ 0.5578.
Poisson PMF for λ = 1.5, 3, 6 — peak shifts with λ
As λ grows, the peak moves right and the curve becomes more symmetric (≈ Normal for λ ≥ 10).
Use cases: calls per hour at a switchboard, accidents per day, decay events per second, defects per metre of cable.
Q4 · Define Normal distribution. State its properties and standardisation rule.8 marks Definition. A continuous RV X follows a Normal distribution with parameters μ (mean) and σ² (variance) if

  2. Mean = Median = Mode = μ.
  3. Total area under the curve = 1.
  4. The X-axis is an asymptote on both sides.
  5. Empirical / 68-95-99.7 rule: ≈ 68 % of data lies in μ ± σ, ≈ 95 % in μ ± 2σ, ≈ 99.7 % in μ ± 3σ.
  6. Standardisation: Z = (X − μ) / σ ⇒ Z ~ N(0, 1).
Standard Normal Curve — 68 / 95 / 99.7 rule
Total area = 1. Z = (X − μ)/σ converts any normal variable to the standard normal table.
