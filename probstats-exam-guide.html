Unit 01 · Random Variables & Distributions

01Random Variables & Distribution Functions

Foundation unit — every exam draws ~25 marks here. Six named distributions + their use-cases dominate Part-B.

High-yield · 25/75 M · 1.5 Q in Part-B

1.1 Probability — Classical, Statistical & Axiomatic always 1.5M

Definition Probability of an event E is the long-run relative frequency with which E occurs in a repeatable random experiment. Formally, P(E) = (favourable outcomes) / (total equally-likely outcomes) for the classical definition.
P(E) ∈ [0,1] P(S) = 1 P(∅) = 0 For mutually exclusive A,B: P(A ∪ B) = P(A) + P(B) For any A,B: P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Three definitions side-by-side

  • Classical (a priori) — outcomes equally likely; assumes a symmetric sample space (coin, die).
  • Statistical (relative frequency) — P(E) = limn→∞ (nE/n); used when symmetry breaks (defective bulbs).
  • Axiomatic (Kolmogorov) — P : F → [0,1] satisfying three axioms: non-negativity, P(S)=1, countable additivity.
How prof asks "Define probability" almost never appears alone — it always feeds into a calculation. Always state the formula + give one numeric example (coin / die / cards). Senior profs subtract 0.5 M for "definition without formula".

1.2 Addition & Multiplication Theorems

Addition: P(A ∪ B) = P(A) + P(B) − P(A ∩ B) Multiplication (independent): P(A ∩ B) = P(A) · P(B) Conditional: P(A|B) = P(A ∩ B) / P(B) (P(B) > 0)
Why it matters The "Two dice — sum divisible by 2 or 3" type appears repeatedly (May-2025, sister papers). Set A=sum divisible by 2, B=sum divisible by 3, then P(A ∪ B) = P(A) + P(B) − P(A ∩ B). Forgetting the intersection costs 4 of 7 marks.

1.3 Conditional Probability & Bayes' Theorem

Bayes': P(Ai|B) = P(B|Ai)·P(Ai) / Σⱼ P(B|Aj)·P(Aj)

Used when prior probabilities are known and we want the posterior probability after observing event B. Classic exam framing: an insurance company has X drivers in three categories with different accident rates — given an accident occurred, what's the probability the driver was a scooter driver? (B.Tech 2023 Q7).

Mnemonic"Prior × Likelihood ÷ Evidence"

1.4 Random Variable — Discrete vs Continuous ⭐ 5×

Definition A random variable X is a real-valued function from the sample space S to ℝ, i.e. X : S → ℝ, which assigns a numerical value to each outcome of a random experiment.
  • Discrete RV — takes countable values (finite or countably infinite). E.g. number of heads in 3 tosses ∈ {0,1,2,3}.
  • Continuous RV — takes any value in an interval. E.g. height of students, lifetime of a bulb.
Discrete: PMF p(xi) = P(X = xi), Σ p(xi) = 1 Continuous: PDF f(x), ∫ f(x) dx = 1, P(X = x) = 0

1.5 Distribution Function (CDF) ⭐ May-2025

Definition The distribution function (CDF) of a random variable X is defined as F(x) = P(X ≤ x) for all x ∈ ℝ. It gives the cumulative probability up to point x.

Four properties (memorize)

  1. Non-decreasing — if x1 ≤ x2, then F(x1) ≤ F(x2).
  2. Bounded — F(−∞) = 0 and F(+∞) = 1.
  3. Right-continuous — limh→0⁺ F(x+h) = F(x).
  4. P(a < X ≤ b) = F(b) − F(a) for any a < b.

1.6 PMF — Probability Mass Function

Definition For a discrete RV X, the PMF p(x) = P(X = x) satisfies p(x) ≥ 0 for all x and Σ p(x) = 1.
F(x) = Σxᵢ ≤ x p(xᵢ) P(a ≤ X ≤ b) = Σa ≤ xᵢ ≤ b p(xᵢ)

1.7 PDF — Probability Density Function ⭐ 6×

Definition For a continuous RV X, the PDF f(x) satisfies (i) f(x) ≥ 0 ∀ x, (ii) ∫−∞∞ f(x) dx = 1, (iii) P(a ≤ X ≤ b) = ∫ab f(x) dx.

Master pattern — when given f(x) = k·g(x) on an interval and asked to verify or find k, integrate over the support and set equal to 1. Solve for k. This pattern appeared in 2022 Q1(a,c), 2023 Q1(a,c), 2024 Q1(i), 2025 Q1(c).

Common pitfall Students forget: for continuous X, P(X = x) = 0. So P(X < a) = P(X ≤ a) — strict and non-strict are interchangeable. Profs deliberately mix these in 5-mark Qs to trap rote-learners.

1.8 Binomial Distribution ⭐ 5×

When to use X = number of successes in n independent Bernoulli trials, each with success probability p. X ~ B(n, p).
PMF: P(X = k) = C(n,k) pk qn−k, k = 0,1,…,n, q = 1−p Mean: E(X) = np Variance: Var(X) = npq MGF: MX(t) = (q + p·et)n

Worked Example (Dec-2024 Q1a)

Given 9·P(X=4) = P(X=2) with n = 6. Find p and P(X=1).

  1. Write both PMFs: 9·C(6,4)p4q2 = C(6,2)p2q4.
  2. C(6,4) = C(6,2) = 15, so 9·15·p4q2 = 15·p2q4 → 9p2 = q2 → 3p = q.
  3. Combined with p + q = 1: p = 1/4, q = 3/4.
  4. P(X=1) = C(6,1)·(1/4)1·(3/4)5 = 6·(1/4)·(243/1024) = 1458/4096 ≈ 0.356.
Trick Whenever an equation links two binomial probabilities, divide out common factorials and solve the resulting equation in p,q.

1.9 Geometric Distribution

When to use X = trial on which the first success occurs in a sequence of Bernoulli trials.
PMF: P(X = k) = qk−1·p, k = 1,2,3,… Mean: E(X) = 1/p Variance: Var(X) = q/p2 MGF: MX(t) = p·et / (1 − q·et), q·et < 1

Memorable property — memoryless: P(X > m+n | X > m) = P(X > n). The only discrete distribution with this property.

1.10 Poisson Distribution ⭐ 4×

When to use X = count of rare events in a fixed interval of time/space with average rate λ. Used when n large, p small, np = λ moderate.
PMF: P(X = k) = e−λ λk / k!, k = 0,1,2,… Mean = Variance = λ MGF: MX(t) = exp(λ(et − 1))

Five conditions (May-2025 Q1b)

  1. Events occur independently in disjoint sub-intervals.
  2. Probability of an event in a small interval Δt is proportional to Δt.
  3. Probability of two or more events in Δt is negligible (o(Δt)).
  4. The rate λ is constant over the entire interval.
  5. n is large (≥ 50) and p is small (≤ 0.1) — approximation to binomial.
Trick — fitting Poisson Step 1: compute λ̂ = Σ(x·f) / N from the data. Step 2: plug into PMF for each k. Step 3: expected freq = N·P(X=k). Appeared 2022 Q6a & 2023 Q6a with identical data (X=0..4, F=192,100,24,3,1).

Quick diff-Q · Binomial vs Poisson

Binomial vs Poisson — when to pick which

1.11 Uniform Distribution (Discrete & Continuous)

Discrete Uniform on {1,2,…,n}

PMF: P(X = k) = 1/n, k = 1,…,n Mean: (n+1)/2 Variance: (n2−1)/12 MGF: MX(t) = (1/n)·Σk=1n etk

Continuous Uniform on [a,b]

PDF: f(x) = 1/(b−a), a ≤ x ≤ b Mean: (a+b)/2 Variance: (b−a)2/12

1.12 Exponential Distribution

When to use X = waiting time until the next event in a Poisson process with rate λ. Models lifetimes, inter-arrival times.
PDF: f(x) = λ·e−λx, x ≥ 0 CDF: F(x) = 1 − e−λx Mean: E(X) = 1/λ Variance: Var(X) = 1/λ2 MGF: MX(t) = λ / (λ − t), t < λ

Memoryless — P(X > s+t | X > s) = P(X > t). The only continuous distribution with this property.

1.13 Normal (Gaussian) Distribution ⭐ 5×

When to use The universal bell curve. By the Central Limit Theorem, sample means of any distribution tend to Normal for large n. X ~ N(μ, σ2).
PDF: f(x) = (1/(σ√(2π))) · exp(−(x−μ)2/(2σ2)) Standardisation: Z = (X − μ)/σ ~ N(0,1) Mean = μ, Variance = σ2, Mode = Median = μ MGF: MX(t) = exp(μt + σ2t2/2)

Three properties (always tested)

  1. Symmetry about x = μ.
  2. Empirical rule — 68% within ±1σ, 95% within ±2σ, 99.7% within ±3σ.
  3. All odd central moments vanish; μ2n = (2n−1)·σ2·μ2n−2.
How prof asks (Dec-2024 Q3a) "Show that for Normal all odd moments vanish, and prove μ2n = (2n−1)σ2μ2n−2." Senior profs love this because it tests symmetry argument and integration-by-parts. State the MGF, expand, equate coefficients of tn/n!.

1.14 Joint Probability Distribution

Definition For two RVs (X,Y), f(x,y) is the joint pdf if f(x,y) ≥ 0 and ∫∫f(x,y)dx dy = 1 over the support.

Five properties (Dec-2024 Q1i & May-2025 Q1e)

  1. F(x,y) is non-decreasing in both arguments.
  2. F(−∞, y) = F(x, −∞) = 0; F(∞, ∞) = 1.
  3. Marginal CDF of X: FX(x) = F(x, ∞); similarly for Y.
  4. For continuous case: f(x,y) = ∂2F/∂x∂y.
  5. X & Y independent ⟺ F(x,y) = FX(x)·FY(y) ⟺ f(x,y) = fX(x)·fY(y).

02Moments & Moment Generating Functions

Theoretical core — expectation, variance, MGF, joint distributions, independence. MGF problems appear every paper.

High-yield · 18/75 M · 1.5 Q in Part-B

2.1 Expectation of a Random Variable

Definition Expectation (or mean) is the "long-run average value" of X. Discrete: E(X) = Σ x·p(x). Continuous: E(X) = ∫ x·f(x) dx.

Four key properties

  1. Linearity — E(aX + bY) = aE(X) + bE(Y), even if X,Y dependent.
  2. Constant — E(c) = c.
  3. Independence — if X ⫫ Y, then E(XY) = E(X)·E(Y).
  4. Function of RV — E(g(X)) = Σ g(x)p(x) or ∫ g(x)f(x)dx.

2.2 Variance & Standard Deviation

Var(X) = E[(X − μ)2] = E(X2) − [E(X)]2 SD(X) = √Var(X) Var(aX + b) = a2·Var(X) Var(X ± Y) = Var(X) + Var(Y) ± 2·Cov(X,Y)
Computational shortcut Always compute E(X2) and E(X) separately, then Var = E(X2) − μ2. Saves time vs (X − μ)2 expansion.

2.3 Raw vs Central Moments

Raw moments (about origin): μ'r = E(Xr) Central moments (about mean): μr = E[(X − μ)r] Relation to raw: μ2 = μ'2 − μ'12 μ3 = μ'3 − 3μ'2μ'1 + 2μ'13 μ4 = μ'4 − 4μ'3μ'1 + 6μ'2μ'12 − 3μ'14
Why this matters Asked verbatim Dec-2024 Q1e & Q6a — "Relation between moment about mean and moment about arbitrary point". State the four formulas. Show derivation for μ2 from binomial expansion of (X−A)2.

2.4 Probability Generating Function (PGF)

Definition For a non-negative integer-valued RV X, GX(s) = E(sX) = Σ sk·P(X=k) for |s| ≤ 1.

Use — GX(1) = 1; G'X(1) = E(X); G''X(1) = E(X(X−1)). PGF uniquely identifies the distribution.

2.5 Moment Generating Function (MGF) ⭐ 5×

Definition MX(t) = E(etX) for all t in a neighbourhood of 0 where the expectation exists. If MGF exists, it uniquely determines the distribution.
MX(t) = E(etX) = 1 + tμ'1 + (t2/2!)μ'2 + (t3/3!)μ'3 + … r-th raw moment: μ'r = (dr/dtr) MX(t) |t=0

Four properties

  1. Uniqueness — If MX(t) = MY(t) on a neighbourhood of 0, then X and Y have the same distribution.
  2. Linear transform — MaX+b(t) = ebt·MX(at).
  3. Sum of independent — MX+Y(t) = MX(t)·MY(t) when X ⫫ Y.
  4. Moments — r-th derivative at t=0 yields r-th raw moment.

Worked Example (May-2025 Q5b · 7 M)

Given f(x) = 1/3 for −1 < x < 2 and 0 otherwise. Find MX(t).

  1. By definition, MX(t) = ∫−12 etx·(1/3) dx.
  2. Integrate: MX(t) = (1/3)·[etx/t]−12 = (1/3t)·(e2t − e−t), valid for t ≠ 0.
  3. At t = 0, L'Hôpital or direct: MX(0) = ∫−12(1/3)dx = 1.
  4. Therefore MX(t) = (e2t − e−t)/(3t) for t ≠ 0, and = 1 at t = 0. ∎

2.6 Two-Dimensional RV — Joint, Marginal, Conditional ⭐ 4×

Joint pdf: f(x,y), ∫∫f(x,y)dx dy = 1 Marginal X: fX(x) = ∫f(x,y) dy Marginal Y: fY(y) = ∫f(x,y) dx Conditional: fY|X(y|x) = f(x,y)/fX(x) Independence: f(x,y) = fX(x)·fY(y)
Exam pattern (3 papers in a row) f(x,y) = (6−x−y)/8, 0 < x < 2, 2 < y < 4. Asked: marginal of X, marginal of Y, conditional, independence test, P(X < 1, Y < 3). Identical question framing 2023 Q3c, 2024 Q4a, 2024 Q4 (12M). Memorise the structure!

2.7 Independence of Random Variables

X and Y are independent iff their joint distribution factors as the product of marginals: f(x,y) = fX(x)·fY(y) for all (x,y).

Test for independence Compute marginals individually, then multiply. If the product matches joint at every point → independent. Even one failing point disproves independence. Don't just check the boundary!

2.8 Covariance & Correlation

Cov(X,Y) = E[(X − μX)(Y − μY)] = E(XY) − E(X)·E(Y) ρ(X,Y) = Cov(X,Y) / (σX·σY), −1 ≤ ρ ≤ 1 X ⫫ Y ⟹ Cov(X,Y) = 0 (converse not generally true)
May-2025 Q1f "Define covariance. What is covariance of two independent variables?" → Cov = 0. Critical caveat: Cov = 0 does NOT imply independence (only for jointly Normal RVs does it).

2.9 Conditional Expectation

E(Y | X = x) = ∫ y·fY|X(y|x) dy Law of total expectation: E(Y) = E[E(Y|X)] Law of total variance: Var(Y) = E[Var(Y|X)] + Var[E(Y|X)]

3.2 Classification & Representation of Data

Classification methods:

  • Geographical — by region/state
  • Chronological — by time (e.g. time series)
  • Qualitative — by attribute (gender, profession)
  • Quantitative — by measurable value (income, age)

Representation methods: Tables (frequency tables, cross-tabs) · Diagrams (bar, pie, histogram) · Graphs (line, ogive, frequency polygon).

3.3 Frequency Distribution

Definition Tabular arrangement of data showing how many times each value or class interval occurs. Two types: discrete (each value) and grouped/continuous (class intervals).

Construction steps: (1) Find range R = max − min. (2) Decide number of classes (k ≈ 1 + 3.322·log₁₀N — Sturges' rule). (3) Class width = R/k. (4) Tally each observation. (5) Tabulate frequencies.

3.4 Mean, Median, Mode — Three Averages ⭐ 4×

Mean (Arithmetic): x̄ = Σf·x / Σf (ungrouped: x̄ = Σx/n) Median (grouped): M = L + ((N/2 − cf)/f)·h Mode (grouped): Z = L + ((f₁ − f₀)/(2f₁ − f₀ − f₂))·h Empirical relation: Mode ≈ 3·Median − 2·Mean

Worked Example (May-2025 Q1g)

Find median of: 20, 25, 30, 15, 17, 35, 26, 18, 40, 45, 50.

  1. Sort ascending: 15, 17, 18, 20, 25, 26, 30, 35, 40, 45, 50. n = 11.
  2. Median position = (n+1)/2 = 6th value.
  3. Median = 26.
Merits/Demerits cheat (Dec-2023 Q1j) Median: Merit — unaffected by extreme values, suitable for open-ended classes. Demerit — not algebraically tractable, ignores values of items. Mode: Merit — easy to locate graphically. Demerit — not unique (bimodal data), unstable for small samples.

3.5 Dispersion — Range · MD · Variance · SD

Range: R = xmax − xmin Mean dev: MD = Σ|x − x̄|/n Variance: σ2 = Σf(x − x̄)2/Σf Standard dev: σ = √σ2 Coeff of var: CV = (σ/x̄)·100 %

SD is the most widely used measure — it's in the same units as the data, is algebraically tractable, and feeds directly into normal-distribution Z-scores.

3.6 Covariance (revisited from Unit 2)

Cov(X,Y) = Σ(x − x̄)(y − ȳ)/n = Σxy/n − x̄·ȳ For independent variables: Cov(X,Y) = 0

3.7 Skewness ⭐ 4×

Definition Skewness measures the asymmetry of a distribution about its mean. Positively skewed: long right tail, mean > median > mode. Negatively skewed: long left tail, mode > median > mean. Symmetric: mean = median = mode.
Karl Pearson: SkP = (Mean − Mode)/SD or SkP = 3(Mean − Median)/SD (when mode ill-defined) range: −3 to +3 Bowley: SkB = (Q₃ + Q₁ − 2·Q₂)/(Q₃ − Q₁), range −1 to +1 Moments: β₁ = μ₃2/μ₂3, γ₁ = √β₁

3.8 Kurtosis — Mesokurtic / Leptokurtic / Platykurtic ⭐ 3×

Definition Kurtosis measures the peakedness (and tail-heaviness) of a distribution compared to the Normal. β₂ = μ₄/μ₂2. γ₂ = β₂ − 3 (excess kurtosis).

4.1 Correlation — Types & Interpretation

Definition Correlation measures the strength and direction of linear association between two variables X and Y. Coefficient r ∈ [−1, +1]; values close to ±1 indicate strong association, 0 indicates no linear association.

Six types (May-2025 Q1j & Dec-2023 Q1i)

  • Positive — both move same direction (height vs weight).
  • Negative / Inverse — move opposite directions (price vs demand).
  • Linear — relation falls on a straight line.
  • Non-linear (Curvilinear) — relation follows a curve.
  • Simple — two variables only (X vs Y).
  • Multiple / Partial — three or more variables (Y with X1, X2, …).

4.2 Karl Pearson's Correlation Coefficient ⭐ 5×

r = Σ(x − x̄)(y − ȳ) / √[Σ(x − x̄)2·Σ(y − ȳ)2] = Cov(X,Y)/(σX·σY) = (NΣxy − ΣxΣy) / √[(NΣx2 − (Σx)2)(NΣy2 − (Σy)2)]

Five properties

  1. −1 ≤ r ≤ +1.
  2. r is dimensionless, independent of units of X and Y.
  3. r is invariant under change of origin and scale.
  4. r is symmetric: r(X,Y) = r(Y,X).
  5. r2 (coefficient of determination) = fraction of variance in Y explained by linear relation with X.
Dec-2024 Q1g shortcut Given Σ(x−x̄)2 = 136, Σ(y−ȳ)2 = 138, Σ(x−x̄)(y−ȳ) = 122. Then r = 122/√(136·138) = 122/√18768 = 122/137.0 ≈ 0.891. Strong positive correlation. ∎

4.3 Spearman's Rank Correlation ⭐ 3×

ρ = 1 − 6·Σd2/(n(n2 − 1)) where di = rank(Xi) − rank(Yi) Tied ranks correction: add (t3 − t)/12 to Σd2 for each tie group of size t

Use when: data is ordinal (ranks) or non-linear monotonic relationship. Same range −1 to +1.

4.4 Regression — Line of Best Fit

Regression of Y on X: y − ȳ = byx(x − x̄) where byx = Σ(x−x̄)(y−ȳ)/Σ(x−x̄)2 = r·σy/σx Regression of X on Y: x − x̄ = bxy(y − ȳ) where bxy = Σ(x−x̄)(y−ȳ)/Σ(y−ȳ)2 = r·σx/σy Relation: r2 = byx·bxy (sign of r = sign of either b)

Properties of regression coefficients (Dec-2024 Q1f)

  1. Same sign — Both byx and bxy have the same sign as r.
  2. Bounded product — byx · bxy = r2 ≤ 1, so if one regression coefficient > 1, the other must be < 1.
  3. Geometric mean — |r| = √(byx · bxy); thus correlation is the geometric mean of the two regression coefficients.
  4. Intersect at (x̄, ȳ) — both regression lines pass through the point of means.

4.6 Curve Fitting — Least Squares Method ⭐ 5×

Principle Find parameters that minimise Σ(yobs − ypred)2. For straight line y = a + bx, set partial derivatives w.r.t. a, b to zero — produces the normal equations.

Straight line y = a + bx

Normal equations: Σy = n·a + b·Σx Σxy = a·Σx + b·Σx2 Solve simultaneously for a, b.

Second-degree parabola y = a + bx + cx2

Three normal equations: Σy = n·a + b·Σx + c·Σx2 Σxy = a·Σx + b·Σx2 + c·Σx3 Σx2y = a·Σx2 + b·Σx3 + c·Σx4

Exponential y = a·bx

Take logs: log y = log a + x·log b Let Y = log y, A = log a, B = log b. Then Y = A + Bx is linear. Apply straight-line normal equations to (x, Y), then back-transform: a = 10A, b = 10B.

Power curve y = a·xb

Take logs: log y = log a + b·log x Let Y = log y, X = log x. Linear in (X,Y). Fit, then a = 10A, b = B.

Define distribution function for a random variable.

The distribution function (CDF) of a random variable X is F(x) = P(X ≤ x), defined for all real x. It is non-decreasing, right-continuous, satisfies F(−∞) = 0 and F(+∞) = 1, and gives the cumulative probability that X does not exceed x. For a discrete RV, F(x) = Σxᵢ≤x p(xᵢ); for a continuous RV, F(x) = ∫−∞x f(t) dt. (See topic 1.5 for diagram.)

What are the conditions under which Poisson distribution is used?

The Poisson distribution applies when (1) events occur independently in disjoint sub-intervals; (2) the probability of an event in a small interval Δt is proportional to Δt; (3) the chance of two or more events in Δt is negligible; (4) the average rate λ is constant over the interval; (5) it serves as the limiting form of B(n,p) when n → ∞, p → 0, np = λ (large n, small p). Examples: number of phone calls per hour at a call centre; defects per metre of cable.

Explain the merits and demerits of Median and Mode.

Median

  • Merits — (1) unaffected by extreme values (outlier-robust); (2) can be located graphically (ogive); (3) suitable for open-ended class intervals; (4) easy to understand.
  • Demerits — (1) not based on all observations; (2) not algebraically tractable; (3) requires sorted data; (4) erratic for small samples.

Mode

  • Merits — (1) easy to locate; (2) useful for categorical/qualitative data; (3) not affected by extreme values; (4) represents the most "typical" value.
  • Demerits — (1) not always unique (bimodal/multimodal); (2) not algebraically tractable; (3) unstable in small samples; (4) ill-defined for symmetric data.

  • |r| ∈ [0, 1]: 0 = none, ±1 = perfect; verify sign of numerator.
  • SkP = (Mean − Mode)/SD; if mode ill-defined use 3(Mean − Median)/SD.
  • Sk = 0 ⇒ symmetric; Sk > 0 ⇒ right-skewed (mean > median > mode).

Joint pdf problems — find marginal, conditional, independence (especially f(x,y) = (6−x−y)/8).

Appeared: 2022 Q3(a), 2023 Q3(c), 2024 Q4(a) (12 M), 2025 Q5(a).

Master sequence (15 M template):

  1. Verify f(x,y) is pdf by integrating over support and checking = 1.
  2. Marginal of X: fX(x) = ∫ f(x,y) dy over y-support.
  3. Marginal of Y: fY(y) = ∫ f(x,y) dx over x-support.
  4. Conditional: f(y|x) = f(x,y)/fX(x).
  5. Independence test: check fX·fY = f(x,y). If equal — independent; otherwise not.
  6. If asked probability — integrate joint over given region.

See PYQ-2024-4a for full 12-M walkthrough of the (6−x−y)/8 example.

Key takeaways · revise these 5
  • Always verify ∬ f(x,y) dx dy = 1 over the full support first — earns the first mark.
  • Marginal of X integrates out y; marginal of Y integrates out x — never swap.
  • Conditional f(y|x) = f(x,y)/fX(x); valid only where fX(x) > 0.
  • Independence test: f(x,y) = fX(x)·fY(y) for all (x,y) in support.
  • For a region probability, integrate joint pdf over the specified region — not the marginals.
