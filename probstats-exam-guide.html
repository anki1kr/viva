1.1 Probability Definitions & Rules 1.5M / 7M

Plain English Probability is a number between 0 and 1 telling us how often an event would happen if we repeated the experiment forever. 0 = never, 1 = always, 0.5 = half the time.
Why it matters Every Part-A 1.5M question on probability begins with a definition request paired with a numeric example. Without the formula, you lose 0.5M instantly.
P(E) = n(E) / n(S) Classical: equally likely outcomes. Statistical: P(E) = lim(n->inf) n_E / n Axiomatic: P(S)=1, P(E)>=0, countable additivity.

Worked Numerical Example: Two-Dice Union (May-2025 Q2b · 7 M)

Two dice are thrown. Find P(sum divisible by 2 or 3).

Sample Space: Total outcomes n(S) = 6 × 6 = 36.
Event A (Sum divisible by 2 / Even): Sums {2, 4, 6, 8, 10, 12}. Favourable outcomes: (1,1), (1,3), (3,1), (2,2), (1,5), (5,1), (2,4), (4,2), (3,3), (2,6), (6,2), (3,5), (5,3), (4,4), (4,6), (6,4), (5,5), (6,6) = 18 outcomes. P(A) = 18/36 = 1/2.
Event B (Sum divisible by 3): Sums {3, 6, 9, 12}. Favourable outcomes: (1,2), (2,1), (1,5), (5,1), (2,4), (4,2), (3,3), (3,6), (6,3), (4,5), (5,4), (6,6) = 12 outcomes. P(B) = 12/36 = 1/3.
Event A ∩ B (Sum divisible by both 2 & 3, i.e., 6): Sums {6, 12}. Favourable outcomes: (1,5), (5,1), (2,4), (4,2), (3,3), (6,6) = 6 outcomes. P(A ∩ B) = 6/36 = 1/6.
Apply Addition Theorem: P(A ∪ B) = P(A) + P(B) − P(A ∩ B) = 1/2 + 1/3 − 1/6 = 3/6 + 2/6 − 1/6 = 4/6 = 2/3 ≈ 0.667.

1.2 Bayes' Theorem & Conditional Probability ⭐ 8M

Plain English Bayes flips the conditional: we know P(symptom | disease), Bayes gives us P(disease | symptom). Use it whenever you have priors + likelihoods and want a posterior.
Prof Pattern 3 of 6 papers · 6-8M Part-B. Always frames as "an event occurred — find probability it came from category X". The 2023 Q7a insurance scenario is the canonical stem.
P(A_i | B) = [ P(B | A_i) * P(A_i) ] / [ Sum_j (P(B | A_j) * P(A_j)) ]

Worked Numerical Example: Scooter/Car/Truck Driver Accidents (University PYQ · 8 M)

An insurance company insured 2000 scooter drivers, 4000 car drivers, and 6000 truck drivers. The probabilities of an accident are 0.01, 0.03, and 0.15 respectively. One of the insured persons meets with an accident. What is the probability that he is a scooter driver?

Define Events: Let S, C, T be the events that the driver is a scooter, car, or truck driver. Let A be the event of meeting with an accident.
Prior Probabilities: Total drivers = 2000 + 4000 + 6000 = 12000. P(S) = 2000 / 12000 = 1/6. P(C) = 4000 / 12000 = 1/3. P(T) = 6000 / 12000 = 1/2.
Likelihoods (Accident Probabilities): P(A | S) = 0.01 = 1/100. P(A | C) = 0.03 = 3/100. P(A | T) = 0.15 = 15/100.
Total Probability of Accident (Evidence) P(A): P(A) = P(S)*P(A|S) + P(C)*P(A|C) + P(T)*P(A|T) P(A) = (1/6)*(1/100) + (1/3)*(3/100) + (1/2)*(15/100) P(A) = 1/600 + 3/300 + 15/200 = 1/600 + 6/600 + 45/600 = 52/600 = 13/150 ≈ 0.0867.
Posterior Probability P(S | A) by Bayes' Theorem: P(S | A) = [ P(S) * P(A|S) ] / P(A) P(S | A) = [ (1/6) * (1/100) ] / [ 52/600 ] = [ 1/600 ] / [ 52/600 ] = 1/52 ≈ 0.0192 (or 1.92%).

1.3 Discrete vs Continuous Random Variables 1.5M / 5M

Plain English A random variable is a numeric translator of real-world outcomes. Discrete variables count things (how many cars pass). Continuous variables measure things (heights, times) along a continuous scale.
Core Differences
Feature Discrete RV Continuous RV
Values Finite or countably infinite {x1, x2, ...} Uncountable infinite values (intervals)
Probability Function Probability Mass Function (PMF): P(X = x) Probability Density Function (PDF): f(x)
Summation Rule Sum over all values: ∑ P(X = x) = 1 Integral over support: ∫ f(x) dx = 1
Point Probability P(X = x) can be positive P(X = x) is strictly 0 at any single point

1.4 Cumulative Distribution Function (CDF) 1.5M

Definition The Cumulative Distribution Function of an RV X is defined as F(x) = P(X <= x) for all x.
Properties of CDF F(x): 1. Bounded: 0 <= F(x) <= 1, F(-inf) = 0, F(inf) = 1. 2. Monotonically Non-decreasing: if a < b, then F(a) <= F(b). 3. Right-continuous: lim(h->0+) F(x + h) = F(x). 4. Probability on Interval: P(a < X <= b) = F(b) - F(a).

1.5 Binomial Distribution ⭐ 8M

Plain English Use Binomial when you toss a coin multiple times or inspect samples with-replacement. You have n independent trials, each with success probability p.
PMF: P(X = k) = C(n, k) * p^k * q^(n-k), where q = 1 - p Mean: np Variance: npq MGF: M_X(t) = (q + p*e^t)^n

Worked Numerical Example: Expected Child Families (Dec-2024 Q2a verbatim · 8 M)

Find the expected number of families having: (a) 2 boys and 2 girls, (b) at least 1 boy, (c) no girls, (d) at most 2 girls, out of 800 families with 4 children each. Assume equal probability for boys and girls.

Identify Parameters: n = 4 children per family. p = 0.5 (prob of boy), q = 0.5 (prob of girl). Total families N = 800. Let X be the number of boys in a family. X ~ B(n=4, p=0.5). PMF: P(X = k) = C(4, k) * (0.5)^k * (0.5)^(4-k) = C(4, k) / 16.
(a) 2 boys and 2 girls (X = 2): P(X = 2) = C(4, 2) / 16 = 6/16 = 3/8. Expected families = N * P(X = 2) = 800 * (3/8) = 300 families.
(b) At least 1 boy (X >= 1): P(X >= 1) = 1 − P(X = 0) = 1 − [ C(4, 0) / 16 ] = 1 − 1/16 = 15/16. Expected families = N * P(X >= 1) = 800 * (15/16) = 750 families.
(c) No girls (i.e., all 4 boys, X = 4): P(X = 4) = C(4, 4) / 16 = 1/16. Expected families = N * P(X = 4) = 800 * (1/16) = 50 families.
(d) At most 2 girls (i.e., 2, 3, or 4 boys, X >= 2): P(X >= 2) = P(X=2) + P(X=3) + P(X=4) = 6/16 + 4/16 + 1/16 = 11/16. Expected families = N * P(X >= 2) = 800 * (11/16) = 550 families.

1.6 Poisson Distribution & Fit Poisson ⭐ 15M

Plain English Poisson models rare events occurring in a fixed interval of space or time (e.g., website clicks per min, network faults, typos).
PMF: P(X = k) = [ e^(-L) * L^k ] / k! where L is the mean rate (lambda) Mean: L Variance: L MGF: M_X(t) = exp(L * (e^t - 1))

Worked Numerical Example: Fitting Poisson to Death Data (University PYQ · 15 M)

Fit a Poisson distribution to the following data of deaths per day in an army camp and calculate theoretical frequencies:

Deaths (x) 0 1 2 3 4
Frequency (f) 122 60 15 2 1
Calculate Mean rate (L): Total Frequency N = 122 + 60 + 15 + 2 + 1 = 200. Sum fx = (0 × 122) + (1 × 60) + (2 × 15) + (3 × 2) + (4 × 1) = 0 + 60 + 30 + 6 + 4 = 100. Mean L = Sum fx / N = 100 / 200 = 0.5.
Compute initial probability e^(-0.5): e^(-0.5) ≈ 0.6065.
Formulate Theoretical Probabilities & Frequencies (T_x = N * P(X = x)): - P(X = 0) = e^(-0.5) = 0.6065 => T_0 = 200 * 0.6065 = 121.3 - P(X = 1) = L * P(X = 0) = 0.5 * 0.6065 = 0.30325 => T_1 = 200 * 0.30325 = 60.65 - P(X = 2) = (L/2) * P(X = 1) = (0.5/2) * 0.30325 = 0.07581 => T_2 = 200 * 0.07581 = 15.16 - P(X = 3) = (L/3) * P(X = 2) = (0.5/3) * 0.07581 = 0.01264 => T_3 = 200 * 0.01264 = 2.53 - P(X = 4) = (L/4) * P(X = 3) = (0.5/4) * 0.01264 = 0.00158 => T_4 = 200 * 0.00158 = 0.32
Final Tabulation & Verification: Theoretical frequencies rounded: {121, 61, 15, 3, 0}. Sum of theoretical frequencies = 121.3 + 60.65 + 15.16 + 2.53 + 0.32 = 199.96 ≈ 200 ✓.

Poisson Approximation to Binomial Worked Example (Dec-2024 Q2b · 7 M)

Given razor blade packets of 10, consignment N = 10,000 packets. Prob of a defective blade is p = 0.002. Using Poisson, find number of packets with (a) no defective, (b) 1 defective, (c) 2 defective blades.

Parameters: n = 10 blades per packet. p = 0.002. Mean L = np = 10 × 0.002 = 0.02. Consignment N = 10,000. P(X = k) = e^(-0.02) * (0.02)^k / k! e^(-0.02) ≈ 0.9802.
(a) No defectives (X = 0): P(X = 0) = 0.9802. Expected packets = N * P(X = 0) = 10000 * 0.9802 = 9802 packets.
(b) 1 defective (X = 1): P(X = 1) = e^(-0.02) * 0.02 = 0.9802 * 0.02 = 0.019604. Expected packets = N * P(X = 1) = 10000 * 0.019604 = 196 packets.
(c) 2 defectives (X = 2): P(X = 2) = e^(-0.02) * (0.02)^2 / 2 = 0.9802 * 0.0004 / 2 = 0.000196. Expected packets = N * P(X = 2) = 10000 * 0.000196 = 2 packets.

1.7 Normal Distribution ⭐ 8M

Plain English The bell curve. Almost all natural continuous variations (physical measurements, IQ, battery life) follow Normal. Scale variable using standard normal Z-score, then check Φ-table.
PDF: f(x) = [ 1 / (S * sqrt(2*pi)) ] * exp( -0.5 * ((x - M)/S)^2 ) Standard Z-Score: Z = (X - M) / S (where M = mean, S = SD) Mean: M, Variance: S^2, MGF: exp(M*t + 0.5 * S^2 * t^2)

Worked Numerical Example: Life of Battery Cells (May-2025 Q3a verbatim · 8 M)

The life of battery cells is normally distributed with a mean of 12 hours and SD of 3 hours. Find the percentage of cells having life: (i) more than 15 hours, (ii) less than 6 hours, (iii) between 10 and 14 hours.

Setup Standardisation: Mean M = 12, SD S = 3. Let X be the life of a cell. Z = (X − 12) / 3.
(i) More than 15 hours (X > 15): Z_15 = (15 − 12) / 3 = 1.00. P(X > 15) = P(Z > 1) = 1 − Φ(1.0) = 1 − 0.8413 = 0.1587 = 15.87%.
(ii) Less than 6 hours (X < 6): Z_6 = (6 − 12) / 3 = −2.00. P(X < 6) = P(Z < −2) = P(Z > 2) = 1 − Φ(2.0) = 1 − 0.9772 = 0.0228 = 2.28%.
(iii) Between 10 and 14 hours (10 < X < 14): Z_10 = (10 − 12) / 3 = −0.67. Z_14 = (14 − 12) / 3 = +0.67. P(10 < X < 14) = P(−0.67 < Z < 0.67) = Φ(0.67) − Φ(−0.67) Φ(0.67) ≈ 0.7486. Φ(−0.67) = 1 − Φ(0.67) = 0.2514. P(10 < X < 14) = 0.7486 − 0.2514 = 0.4972 = 49.72%.

2.1 Expectation & Moments Transformation ⭐ 8M

Plain English Expectation is the theoretical average of a random variable. A moment measures the shape of the distribution (spread, skew, pointiness). Raw moments are measured about an arbitrary point A; central moments are measured about the mean.
E(X) = ∑ x * p(x) (Discrete) E(X) = ∫ x * f(x) dx (Continuous) r-th Raw Moment: u_r' = E[(X - A)^r] r-th Central Moment: u_r = E[(X - Mean)^r]

Worked Numerical Example: Raw-to-Central Moments (Dec-2024 Q5a verbatim · 8 M)

The first four raw moments of a distribution about the value A = 4 are -1.5, 17, -30, and 108. Find the moments about the mean. Also determine skewness (β₁) and kurtosis (β₂).

Identify Given Values: A = 4. u_1' = -1.5, u_2' = 17, u_3' = -30, u_4' = 108.
Find True Mean: Mean (u) = A + u_1' = 4 + (-1.5) = 2.5.
Compute Central Moments (u₂ and u₃): - u_2 = u_2' − (u_1')^2 = 17 − (-1.5)^2 = 17 − 2.25 = 14.75. - u_3 = u_3' − 3*u_2'*u_1' + 2*(u_1')^3 u_3 = -30 − 3*(17)*(-1.5) + 2*(-1.5)^3 u_3 = -30 + 76.5 + 2*(-3.375) = -30 + 76.5 − 6.75 = 39.75.
Compute Central Moment (u₄): u_4 = u_4' − 4*u_3'*u_1' + 6*u_2'*(u_1')^2 − 3*(u_1')^4 u_4 = 108 − 4*(-30)*(-1.5) + 6*(17)*(-1.5)^2 − 3*(-1.5)^4 u_4 = 108 − 180 + 6*(17)*(2.25) − 3*(5.0625) u_4 = 108 − 180 + 229.5 − 15.1875 = 142.3125.
Calculate Skewness (β₁) and Kurtosis (β₂): - Skewness coefficient: β₁ = (u_3)^2 / (u_2)^3 = (39.75)^2 / (14.75)^3 = 1580.0625 / 3209.64 ≈ 0.492. - Kurtosis coefficient: β₂ = u_4 / (u_2)^2 = 142.3125 / (14.75)^2 = 142.3125 / 217.5625 ≈ 0.654. Interpretation: Since β₁ > 0, the distribution is positively skewed (right-skewed). Since β₂ < 3, it is platykurtic (flatter than a normal distribution).

2.2 Moment Generating Function (MGF) 7M / 8M

Plain English MGF is like a zip file of all moments. Instead of calculating ∑x^r * p(x) one by one, you find M_X(t), differentiate it r times at t=0, and get the r-th raw moment.
M_X(t) = E[e^(tX)] Discrete: ∑ e^(t*x_i) * p(x_i) Continuous: ∫ e^(t*x) * f(x) dx Raw Moment formula: u_r' = [ d^r/dt^r M_X(t) ] at t=0

Worked Numerical Example: MGF of Triangular PDF (University PYQ · 8 M)

Find the MGF of a random variable X with triangular pdf: f(x) = x for 0 <= x < 1, and f(x) = 2 - x for 1 <= x < 2.

Setup Integral: M_X(t) = ∫₀¹ x * e^(tx) dx + ∫₁² (2 - x) * e^(tx) dx.
Evaluate First Part (I₁ = ∫₀¹ x * e^(tx) dx): Using integration by parts: u = x, dv = e^(tx) dx. I₁ = [ x * e^(tx) / t ]₀¹ − ∫₀¹ (e^(tx) / t) dx I₁ = [ e^t / t ] − [ e^(tx) / t^2 ]₀¹ = e^t / t − e^t / t^2 + 1 / t^2.
Evaluate Second Part (I₂ = ∫₁² (2 - x) * e^(tx) dx): Using parts: u = 2-x, dv = e^(tx) dx. I₂ = [ (2-x) * e^(tx) / t ]₁² − ∫₁² ( -e^(tx) / t ) dx I₂ = [ 0 − e^t / t ] + [ e^(tx) / t^2 ]₁² = −e^t / t + e^(2t) / t^2 − e^t / t^2.
Combine parts: M_X(t) = I₁ + I₂ = (e^t / t − e^t / t^2 + 1 / t^2) + (−e^t / t + e^(2t) / t^2 − e^t / t^2) M_X(t) = ( e^(2t) − 2*e^t + 1 ) / t^2 = (e^t − 1)^2 / t^2.

2.3 Two-Dimensional RVs: Joint, Marginal & Conditional ⭐ 12M / 15M

Plain English When you measure two traits per subject (e.g., height X and weight Y), they form a two-dimensional variable. Marginal distributions show each trait alone. Conditional distributions show one trait's odds given the other trait's exact value.
Verify PDF: ∬ f(x, y) dx dy = 1 Marginal of X: f_X(x) = ∫ f(x, y) dy Marginal of Y: f_Y(y) = ∫ f(x, y) dx Conditional Y|X: f(y | x) = f(x, y) / f_X(x) Independence Test: X ⫫ Y iff f(x, y) = f_X(x) * f_Y(y)

Worked Numerical Example: f(x,y) = (6 - x - y)/8 (Dec-2024 Q4a verbatim · 12 M)

Given the joint PDF f(x,y) = (6 - x - y)/8 for 0 < x < 2 and 2 < y < 4, and 0 elsewhere. Find: (i) P(X < 1, Y < 3), (ii) Marginal PDFs of X and Y, (iii) Conditional PDF f(y|x), (iv) check if X and Y are independent.

(i) Compute probability P(X < 1, Y < 3): Integrate over x from 0 to 1, and y from 2 to 3. P = (1/8) * ∫₀¹ [ ∫₂³ (6 - x - y) dy ] dx Inner integral over y: ∫₂³ (6 - x - y) dy = [ (6 - x)y − y^2/2 ]₂³ = [ 3(6 - x) − 4.5 ] − [ 2(6 - x) − 2 ] = 18 − 3x − 4.5 − 12 + 2x + 2 = 3.5 − x. Outer integral over x: P = (1/8) * ∫₀¹ (3.5 - x) dx = (1/8) * [ 3.5x − x^2/2 ]₀¹ P = (1/8) * (3.5 − 0.5) = (1/8) * 3 = 3/8 = 0.375.
(ii) Find Marginal PDF of X, f_X(x): Integrate out y from 2 to 4. f_X(x) = ∫₂⁴ [ (6 - x - y)/8 ] dy = (1/8) * [ (6 - x)y − y^2/2 ]₂⁴ f_X(x) = (1/8) * ( [ 4(6 - x) − 8 ] − [ 2(6 - x) − 2 ] ) f_X(x) = (1/8) * ( 24 − 4x − 8 − 12 + 2x + 2 ) = (1/8) * (6 − 2x) = (3 − x)/4, for 0 < x < 2.
(ii) Find Marginal PDF of Y, f_Y(y): Integrate out x from 0 to 2. f_Y(y) = ∫₀² [ (6 - x - y)/8 ] dx = (1/8) * [ (6 - y)x − x^2/2 ]₀² f_Y(y) = (1/8) * ( 2(6 - y) − 2 ) = (1/8) * (12 − 2y − 2) = (1/8) * (10 − 2y) = (5 − y)/4, for 2 < y < 4.
(iii) Find Conditional PDF f(y|x): f(y|x) = f(x,y) / f_X(x) = [ (6 - x - y)/8 ] / [ (3 - x)/4 ] f(y|x) = (6 − x − y) / [2 * (3 − x)], for 2 < y < 4.
(iv) Test Independence: Check if f(x,y) = f_X(x) * f_Y(y). f_X(x) * f_Y(y) = [ (3 - x)/4 ] * [ (5 - y)/4 ] = (3 - x)(5 - y) / 16. Since (3 - x)(5 - y) / 16 ≠ (6 - x - y)/8, the variables X and Y are NOT independent.

2.4 Covariance & Correlation of Joint Discrete RVs ⭐ 8M

Plain English Covariance measures the joint variation of two variables. If positive, they grow together. Correlation standardizes covariance into a tidy scale from -1 to +1.
Cov(X,Y) = E(XY) - E(X)*E(Y) r(X,Y) = Cov(X,Y) / [ SD(X) * SD(Y) ] E(XY) = ∑∑ x * y * P(X=x, Y=y)

Worked Numerical Example: Coin Toss Joint PMF (University PYQ · 8 M)

3 fair coins are tossed. Let X = number of heads in the first 2 tosses, Y = number of heads in the last 2 tosses. Compute Cov(X,Y) and correlation coefficient r(X,Y).

List outcomes and map X, Y values: Sample Space (n=8): - HHH: X=2, Y=2 - HHT: X=2, Y=1 - HTH: X=1, Y=1 - HTT: X=1, Y=0 - THH: X=1, Y=2 - THT: X=1, Y=1 - TTH: X=0, Y=1 - TTT: X=0, Y=0
Formulate Joint Probability Distribution P(X, Y): - P(X=0, Y=0) = P(TTT) = 1/8. - P(X=0, Y=1) = P(TTH) = 1/8. - P(X=1, Y=0) = P(HTT) = 1/8. - P(X=1, Y=1) = P(HTH, THT) = 2/8. - P(X=1, Y=2) = P(THH) = 1/8. - P(X=2, Y=1) = P(HHT) = 1/8. - P(X=2, Y=2) = P(HHH) = 1/8. All other combinations (e.g. X=0, Y=2) are 0.
Compute Marginal Expected Values: X can be 0, 1, 2. Marginal P(X): P(X=0) = 2/8, P(X=1) = 4/8, P(X=2) = 2/8. E(X) = 0*(2/8) + 1*(4/8) + 2*(2/8) = 1. By symmetry, E(Y) = 1. Expected squares: E(X^2) = 0*(2/8) + 1^2*(4/8) + 2^2*(2/8) = 4/8 + 8/8 = 1.5. Variance Var(X) = E(X^2) − E(X)^2 = 1.5 − 1.0 = 0.5. By symmetry, Var(Y) = 0.5. SD(X) = SD(Y) = √0.5 ≈ 0.7071.
Compute E(XY): E(XY) = ∑∑ x * y * P(X=x, Y=y) E(XY) = (1×1 × 2/8) + (1×2 × 1/8) + (2×1 × 1/8) + (2×2 × 1/8) E(XY) = 2/8 + 2/8 + 2/8 + 4/8 = 10/8 = 1.25.
Calculate Cov(X,Y) and Correlation r: Cov(X,Y) = E(XY) − E(X)*E(Y) = 1.25 − 1.0 * 1.0 = 0.25. r(X,Y) = Cov(X,Y) / [ SD(X) * SD(Y) ] = 0.25 / (0.7071 * 0.7071) = 0.25 / 0.5 = 0.5. Interpretation: X and Y have a moderate positive correlation. Knowing the first two tosses shares positive information about the last two tosses.

3.1 Central Tendency & Means 1.5M / 5M

Plain English Central tendency represents the "center of gravity" of your dataset. Mean is the algebraic center, Median is the physical middle point, and Mode is the most popular value.
Grouped Mean: Mean = Sum(f * x) / Sum(f) Combined Mean: Mean_c = (n1*Mean1 + n2*Mean2) / (n1 + n2)

Worked Numerical Example: Combined Mean (May-2025 Q3b · 5 M)

A branch has 50 workers earning an average of ₹3000/day, and another branch has 70 workers earning an average of ₹3500/day. Find the combined mean daily wage.

Identify Parameters: n₁ = 50, Mean₁ = 3000. n₂ = 70, Mean₂ = 3500.
Apply Combined Mean Formula: Mean_c = ( n₁ * Mean₁ + n₂ * Mean₂ ) / ( n₁ + n₂ ) Mean_c = ( 50 * 3000 + 70 * 3500 ) / ( 50 + 70 ) Mean_c = ( 150000 + 245000 ) / 120 Mean_c = 395000 / 120 = ₹3291.67.

3.2 Dispersion: Standard Deviation & Variance ⭐ 8M

Plain English SD measures how far, on average, the data points spread out from the mean. Standard deviation is the square root of the variance.
Grouped Variance: s^2 = [ Sum(f * x^2) / N ] - Mean^2 SD: s = sqrt(s^2)

(a) Binomial — Mean, Variance, Mode

The Binomial distribution models the count of "successes" in n independent Bernoulli trials, each with probability p of success and q = 1 − p of failure. Writing X ~ B(n, p):

  • pmf: P(X = k) = C(n, k) · pk · qn−k, k = 0, 1, …, n.
  • Support: X ∈ {0, 1, …, n} — discrete and finite. Mean = np, Variance = npq.
  • Sum of Bernoulli trials: X = Σi=1n Xi where each Xi ~ Bernoulli(p); this representation gives the quickest derivation of E and Var by linearity.
  • Closure under addition: If X ~ B(n1, p) and Y ~ B(n2, p) are independent, X + Y ~ B(n1 + n2, p).
  • MGF: MX(t) = (q + p·et)n.
Derive Mean E(X) = np: E(X) = Σk=0n k · C(n, k) · pk · qn−k. Term k = 0 contributes 0, start at k = 1. Identity: k · C(n, k) = n · C(n − 1, k − 1). E(X) = np · Σk=1n C(n−1, k−1) · pk−1 · qn−k. Let j = k − 1, sum becomes Σj=0n−1 C(n−1, j) pj q(n−1)−j = (p + q)n−1 = 1. ∴ mean = np.
Derive Variance Var(X) = npq: Compute E[X(X−1)] using k(k−1) · C(n, k) = n(n−1) · C(n−2, k−2): E[X(X−1)] = n(n−1) p2 · (p + q)n−2 = n(n−1) p2. E(X2) = E[X(X−1)] + E(X) = n(n−1)p2 + np = n2p2 − np2 + np. Var(X) = E(X2) − [E(X)]2 = n2p2 − np2 + np − n2p2 = np − np2 = np(1 − p) = variance = npq.
Derive Mode: Mode is the k that maximises P(X = k). Examine the ratio: P(X = k) / P(X = k − 1) = [(n − k + 1) / k] · (p / q). Ratio ≥ 1 ⇔ k ≤ (n + 1)p. So P(X = k) increases while k < (n + 1)p, then decreases. ∴ Mode = ⌊(n + 1)p⌋. If (n + 1)p is an integer, both (n + 1)p and (n + 1)p − 1 are modes (bimodal).

(a) Karl Pearson Correlation Coefficient

The Karl Pearson r coefficient measures the strength and direction of linear association between two paired variables, fitted via the method of least squares. Key properties:

  • Range: −1 ≤ r ≤ +1; r = ±1 ⇔ all (xi, yi) lie exactly on a straight line.
  • Unit-free: r is invariant under change of origin and positive scale: r(aX + b, cY + d) = sign(ac) · r(X, Y).
  • Symmetry: rxy = ryx (only one number, unlike regression).
  • Sign: matches the sign of Cov(X, Y) and of both regression slopes.
  • Limitation: measures only linear association; r ≈ 0 does not exclude strong non-linear patterns (e.g. y = x2 on a symmetric x-range).

Two equivalent computational forms:

(deviation form) r = Σ(x − x̄)(y − ȳ) / √[Σ(x − x̄)2 · Σ(y − ȳ)2]
(raw-score form) r = [n·ΣXY − ΣX·ΣY] / √{[n·ΣX2 − (ΣX)2][n·ΣY2 − (ΣY)2]}

(b) Bayes' theorem — Disease Test

Bayes' theorem converts a forward (model → data) probability into a backward (data → model) inference: posterior P(D | +) = P(+ | D) P(D) / P(+), where the prior is P(D) and P(+) is reconstructed from total probability over the partition {D, D′}. The two diagnostic-test inputs are sensitivity P(+ | D) and specificity P(− | D′); the false-positive rate is 1 − specificity.

State Bayes' formula: P(D | +) = [P(+ | D) · P(D)] / P(+). Total probability: P(+) = P(+ | D) · P(D) + P(+ | D′) · P(D′).
Substitute given values: prior P(D) = 0.01 ⇒ P(D′) = 0.99. P(+ | D) = 0.99 (sensitivity). P(+ | D′) = 1 − 0.95 = 0.05 (false-positive rate = 1 − specificity). P(+) = 0.99 × 0.01 + 0.05 × 0.99 = 0.0099 + 0.0495 = 0.0594.
Posterior: P(D | +) = 0.0099 / 0.0594 = P(D | +) = 0.1667 (≈ 16.67%). Interpretation: even after a positive test, the chance of disease is only ~17% because the prior prevalence (1%) is very low: base-rate fallacy in action.

(a) Spearman rank correlation

Spearman rank correlation ρs is the non-parametric rank-based Karl Pearson coefficient computed on the ranks of X and Y rather than their raw values. Properties:

  • Formula (no ties): ρs = 1 − 6·ΣD2 / [n(n2 − 1)], where D = RX − RY.
  • Range: −1 ≤ ρs ≤ +1; ρs = +1 ⇔ ranks are identical, ρs = −1 ⇔ exactly reversed.
  • Captures monotone (not just linear) relationships: handles "y rises faster than x" curves where Pearson r underestimates.
  • Robust to outliers: only relative order matters; an extreme value occupies one rank slot.
  • Tie correction: when ranks tie, replace each tied set with the average of their positions and add the correction Σ(tj3 − tj)/12 to ΣD2.

(b) Joint pdf — Region Probabilities

For a joint pdf f(x, y), the probability of any measurable region R is P((X, Y) ∈ R) = ∫∫R f(x, y) dx dy. When f(x, y) factorises into g(x)·h(y) and the region is a Cartesian product [a, b] × [c, d], the double integral splits into a product of two single integrals: major shortcut. For non-rectangular regions (e.g. X + Y < 1), the inner limits depend on the outer variable and must be derived from the boundary equation.

Verify f(x, y) = 4xy is a valid pdf: Non-negative on [0, 1]2. Total mass: ∫01∫01 4xy dx dy = 4 · [x2/2]01 · [y2/2]01 = 4 · ½ · ½ = 1 ✓. Note the product structure: f(x, y) = (2x)·(2y) ⇒ X and Y are independent with marginal pdfs fX(x) = 2x and fY(y) = 2y on [0, 1].
(i) P(X < ½ ∩ Y < ½): = ∫01/2∫01/2 4xy dx dy = 4 · [x²/2]01/2 · [y²/2]01/2 = 4 · (1/8) · (1/8) = 4 / 64 = P = 1/16 = 0.0625.
(ii) P(X + Y < 1): Triangular region 0 ≤ x ≤ 1, 0 ≤ y ≤ 1 − x. P = ∫01∫01−x 4xy dy dx = ∫01 4x · [y²/2]01−x dx = ∫01 2x(1 − x)2 dx. Expand: 2x(1 − 2x + x²) = 2x − 4x² + 2x³. ∫01 (2x − 4x² + 2x³) dx = [x² − 4x³/3 + x⁴/2]01 = 1 − 4/3 + 1/2 = 6/6 − 8/6 + 3/6 = 1/6. ∴ P(X + Y < 1) = 1/6 ≈ 0.1667.

Pitfalls:
  • For P(X + Y < 1), forgetting that the upper Y-limit depends on x (i.e. integrating 0 → 1 instead of 0 → 1 − x) yields the wrong answer 1. Always sketch the region first.
  • Spearman ρs requires that both R1 and R2 be valid permutations of 1, …, n — students sometimes leave the raw marks in one column and ranks in the other, producing nonsense.
  • The formula 1 − 6ΣD2/[n(n2 − 1)] assumes no ties; with ties, use the tie-corrected version or compute Karl Pearson r directly on the average-ranks.
  • Don't confuse "rank correlation" with "ordinal correlation in general" — Spearman specifically measures monotone agreement of ranks.
Modern Relevance

Spearman's ρs is the non-parametric rank correlation deployed in sports analytics (season-to-season player-ranking stability), Likert-scale survey analysis, judge-vs-judge inter-rater reliability, and feature ranking in monotone-but-non-linear ML pipelines. Joint-pdf region integration is the workhorse of Bayesian credible-region computation, Markov-chain Monte Carlo (MCMC) target evaluation, multivariate risk modelling (joint loss exceedance probabilities), and physics-of-failure reliability engineering where failure boundaries are defined by inequalities on multiple stress variables.

Conclusion

Spearman ρs = 0.9394 indicates the two judges agreed strongly on ranking (within ±1 in every pair); cross-verified by applying Karl Pearson r to the ranks themselves, both methods give 0.9394 to four decimal places. Joint-pdf integrals split into product form when the region is rectangular (P(X < ½ ∩ Y < ½) = 1/16) and require dependent limits when the region is triangular (P(X + Y < 1) = 1/6). Sketching the integration region first prevents both limit errors and sign errors.
