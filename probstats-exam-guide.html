1.1 Probability вҖ” Classical, Statistical & Axiomatic always 1.5M

Definition Probability of an event E is the long-run relative frequency with which E occurs in a repeatable random experiment. Formally, P(E) = (favourable outcomes) / (total equally-likely outcomes) for the classical definition.
P(E) вҲҲ [0,1] P(S) = 1 P(вҲ…) = 0 For mutually exclusive A,B: P(A вҲӘ B) = P(A) + P(B) For any A,B: P(A вҲӘ B) = P(A) + P(B) вҲ’ P(A вҲ© B)

  • Classical (a priori) вҖ” outcomes equally likely; assumes a symmetric sample space (coin, die).
  • Statistical (relative frequency) вҖ” P(E) = limnвҶ’вҲһ (nE/n); used when symmetry breaks (defective bulbs).
  • Axiomatic (Kolmogorov) вҖ” P : F вҶ’ [0,1] satisfying three axioms: non-negativity, P(S)=1, countable additivity.
How prof asks "Define probability" almost never appears alone вҖ” it always feeds into a calculation. Always state the formula + give one numeric example (coin / die / cards). Senior profs subtract 0.5 M for "definition without formula".
Plain English Probability is a number between 0 and 1 telling us how often an event would happen if we repeated the experiment forever. 0 = never, 1 = always, 0.5 = half the time.
Why it matters Every Part-A 1.5 M definition prompt is paired with a numeric example (coin / die / card). Stating only the words without P(E) = favourable/total costs you 0.5 M instantly.

4.1 Correlation вҖ” Types & Interpretation

Definition Correlation measures the strength and direction of linear association between two variables X and Y. Coefficient r вҲҲ [вҲ’1, +1]; values close to Вұ1 indicate strong association, 0 indicates no linear association.

Six types (May-2025 Q1j & Dec-2023 Q1i)

  • Positive вҖ” both move same direction (height vs weight).
  • Negative / Inverse вҖ” move opposite directions (price vs demand).
  • Linear вҖ” relation falls on a straight line.
  • Non-linear (Curvilinear) вҖ” relation follows a curve.
  • Simple вҖ” two variables only (X vs Y).
  • Multiple / Partial вҖ” three or more variables (Y with X1, X2, вҖҰ).

4.2 Karl Pearson's Correlation Coefficient вӯҗ 5Г—

r = ОЈ(x вҲ’ xМ„)(y вҲ’ Иі) / вҲҡ[ОЈ(x вҲ’ xМ„)2В·ОЈ(y вҲ’ Иі)2] = Cov(X,Y)/(ПғXВ·ПғY) = (NОЈxy вҲ’ ОЈxОЈy) / вҲҡ[(NОЈx2 вҲ’ (ОЈx)2)(NОЈy2 вҲ’ (ОЈy)2)]
