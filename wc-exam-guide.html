GSM System Architecture ⭐ Repeated 5×

Definition GSM (Global System for Mobile communications) is the 2G standard separating operations into three major subsystems connected by standard interfaces.

Every 15-mark GSM architecture answer is structured around three logical groups:

Subsystem GroupConstituent ModulesRole in Network
Mobile Station (MS)Handset + SIM CardInsulates subscriber identity. SIM stores IMSI and A3/A8 security keys.
Base Station Subsystem (BSS)BTS (Transceiver) + BSC (Controller)Handles radio link connection (Um interface) and local channel allocation.
Network Switching Subsystem (NSS)MSC (Switch) + HLR + VLR + AuCThe brain. Handles routing, billing, user location caching, and authentication triplets.
The AuC Security Triplet Authentication Centre (AuC) generates **RAND, SRES, and Kc** (the security triplet) using A3 and A8 algorithms, preventing clone phones from stealing network resources.

Bluetooth, Piconets & Scatternets ⭐ Repeated 3×

Bluetooth structures close-range devices using Frequency Hopping Spread Spectrum (FHSS) to prevent interference:

  • Piconet: A local network containing up to 8 active devices: 1 Master (boss) coordinating synchronization + up to 7 Slaves.
  • Scatternet: Formed when a slave in one piconet acts as a bridge (or master) in an adjacent piconet, chaining them together.

Frequency Reuse & Cluster Math ⭐ Guaranteed 10M

Frequency Reuse Low-power antennas allow the same radio channels to be reused in faraway cells. Nearby cells are grouped into a **Cluster (N)**. Each cell inside a cluster gets a distinct set of channels.
Frequency Reuse Formulas
• Q = D / R = sqrt(3N) (Co-channel reuse ratio)
• Standard N sizes = i² + ij + j² (Common sizes: 3, 4, 7, 9, 12)

A smaller cluster size $N$ increases overall network capacity but reduces the physical distance between co-channel cells, increasing **Co-channel Interference (CCI)**.

Cellular Handoff MAHO / MCHO / NCHO ⭐ Repeated 6×

A handoff transfers an active call from one radio channel/base station to another as the subscriber crosses cell boundaries:

    03Multiple Access &
    Spread Spectrum

    Sharing the sky safely using time slots, frequency slots, and unique code mathematics.

    Multiple Access: FDMA, TDMA, CDMA, SDMA ⭐ Repeated 4×

    Access Schemes Mnemonic • FDMA: Separate users by frequency (one channel per user).
    • TDMA: Separate users by time slots (all share one frequency in turns).
    • CDMA: Separate users by orthogonal mathematical codes (all share same band simultaneously).
    • SDMA: Separate users by pointing narrow spatial antenna beams.
    Near-Far Problem in CDMA If a mobile user is physically close to the base station, its signal will completely drown out distant users on the same frequency. CDMA resolves this by enforcing **Strict Power Control** loops (checking and adjusting mobile power 800 times a second).

    Spread Spectrum: DSSS vs FHSS ⭐ Repeated 2×

    Spread Spectrum (SS) Intentionally spreading a narrow-band data signal across a much wider radio frequency band. This reduces interference susceptibility and enables anti-jamming.
    • DSSS (Direct Sequence): Multiplies data bits by a fast Pseudo-Noise (PN) chip code. Low probability of intercept. Used in GPS.
    • FHSS (Frequency Hopping): Carrier frequency hops rapidly across many channels according to a PN sequence. Outstanding anti-interference. Used in Bluetooth.
    Unit IV

    04Wireless Networking &
    Routing Architectures

    Network routing, ISDN architectural channels, and intelligent systems.

    ISDN & AIN Architectures

    • ISDN (Integrated Services Digital Network): Integrates voice and data over standard phone lines. Channels: B Channel (Bearer, 64 kbps, carries voice/data) and D Channel (Delta, 16 kbps, handles out-of-band signaling).
    • AIN (Advanced Intelligent Network): Decouples call-routing databases from physical switches. Allows rapid deployment of custom services (toll-free numbers, call forwarding) without updating local exchange software.

    Q1(a)1.5M
    What is WLL (Wireless Local Loop)? What is its main advantage?
    WLL is a system connecting fixed telephone subscribers to the local exchange via a short wireless link rather than physical copper drops. Advantage: Extremely fast and low-cost deployment in rural or developing markets.
    Q1(b)1.5M
    Explain frequency reuse. Define co-channel cells.
    Frequency reuse is the cellular design technique where low-power antennas reuse the same frequency channels at spatially separated cells. Cells that share the same set of frequencies are called **co-channel cells**.
    Q1(c)1.5M
    List three advantages of digital cellular systems (2G) over analog (1G).
    (1) Higher spectral capacity via TDMA/CDMA packing. (2) Secure encrypted calls. (3) Ability to handle digital SMS and low-rate data services.
    Q1(d)1.5M
    Explain Grade of Service (GoS) in cellular networks. What is the standard target?
    GoS is a metric defining the probability that a call is blocked during peak busy hours. The standard carrier target is GoS ≤ 2% (meaning ≤ 2 out of 100 calls are blocked).
    Q1(e)1.5M
    Differentiate hard handoff and soft handoff.
    Hard Handoff: "Break-before-make" — drops connection with old base station before connecting with the new one. Soft Handoff: "Make-before-break" — retains connection with both stations simultaneously (requires CDMA).
    Q1(f)1.5M
    Define CDMA. What is the processing gain formula?
    CDMA is a multiplexing access scheme where multiple users share the same frequency band simultaneously, separated by unique orthogonal code sequences. Processing Gain formula: G_p = Chip Rate / Data Rate.
    Q1(g)1.5M
    List the three GSM security authentication triplets generated by the AuC.
    The AuC security triplet consists of: (1) RAND (Random number), (2) SRES (Signed Response), and (3) Kc (Cipher key).
    Q1(h)1.5M
    Define scatternet in Bluetooth.
    A scatternet is formed when multiple Bluetooth piconets are linked together, sharing at least one common bridge device acting as a slave in one piconet and a master/slave in another.
    Q1(i)1.5M
    Explain the near-far problem in TDMA and CDMA systems.
    The near-far problem occurs when a mobile unit physically close to the base station transmits at a power level that completely drowns out weak signals from distant users. Enforced power control loops are required to resolve this in CDMA.
    Q1(j)1.5M
    What is the pure ALOHA maximum theoretical throughput and slotted ALOHA throughput?
    Pure ALOHA maximum throughput = 18.4% (at G = 0.5). Slotted ALOHA maximum throughput = 36.8% (at G = 1.0) because forced clock-slot boundaries halve the collision vulnerability window.

    Introduction

    GSM (Global System for Mobile communications) is the canonical 2G digital cellular standard. Its architecture decouples components into three main subsystems connected by standard interfaces (Um, Abis, A), ensuring multi-vendor hardware compatibility.

    The Three Subsystem Groups
    Subsystem GroupNetwork ModuleRole & Function
    Mobile Station (MS)SIM Card + HandsetSIM card stores the IMSI, authentication keys, and user profile data, insulating device hardware from subscription status.
    Base Station Subsystem (BSS)BTS (Base Transceiver Station)Houses radio transmitters/receivers, handles modulation/demodulation and the Um air interface.
    BSC (Base Station Controller)Manages radio channel allocation, frequency hopping, and local handover execution across multiple BTS towers.
    Network Switching Subsystem (NSS)MSC (Mobile Services Switching Centre)The core routing engine. Coordinates call setup, routing switching, billing, and handoffs across different controller zones.
    HLR (Home Location Register)The permanent master database storing subscriber profiles, billing details, and current location registers.
    VLR (Visitor Location Register)A dynamic local cache database storing temporary profiles of guest mobile users currently active in the MSC zone.
    AuC (Authentication Centre)Generates RAND, SRES, and Kc keys using A3/A8 algorithms, providing the MSC with security verification.
    Examiner Trap Do not mix up HLR and VLR. HLR is the permanent database owned by your home network; VLR is a temporary local registry cache holding guest user data to avoid redundant querying back to the home HLR.

    Q3Unit 2 Numerical15M
    A total bandwidth of 25 MHz is allocated to a cellular system with N=7 frequency reuse cluster. If each simplex channel is 25 kHz (duplex pair = 50 kHz), and 1 MHz is reserved for control channels:
    (a) Find the total duplex channels in the system.
    (b) Compute voice & control channels per cell.
    (c) If cell radius R = 2 km, find co-channel reuse ratio Q and co-channel cell distance D.
    (a) Total Duplex Channels
    Step 1
    Compute duplex channel width: each call requires one uplink and one downlink simplex channel. Duplex size = 2 × 25 kHz = 50 kHz = 0.05 MHz.
    Step 2
    Compute total system duplex channels: Total Channels = 25 MHz / 0.05 MHz = 500 duplex channels.
    (b) Voice & Control Channels per Cell
    Step 3
    Compute system-wide control channels from reserved bandwidth: 1 MHz / 0.05 MHz = 20 control channels.
    Step 4
    Compute system-wide voice channels: 500 total − 20 control = 480 voice channels.
    Step 5
    Compute voice and control channels per cell (using cluster size N = 7):
    • Voice channels per cell = 480 / 7 ≈ 68.57 → 68 channels per cell (rounded down).
    • Control channels per cell = 20 / 7 ≈ 2.86 → 3 channels per cell (rounded up).
    (c) Reuse Ratio Q & Co-channel Distance D
    Step 6
    Compute co-channel reuse ratio Q: Q = sqrt(3 N) = sqrt(21) ≈ 4.58.
    Step 7
    Compute co-channel cell distance D (using cell radius R = 2 km): D = Q × R = 4.58 × 2 km = 9.16 km.

    Answers summary: (a) 500 duplex channels, (b) 68 voice & 3 control channels/cell, (c) Q ≈ 4.58, D ≈ 9.16 km.

    Q4Unit 2 Highlight15M
    Compare MAHO, MCHO, and NCHO handoff strategies. Discuss the practical handoff considerations: hysteresis, dwell time, guard channels, and the ping-pong effect. Trace the 4-stage handoff sequence.
    Handoff Strategies Comparison
    Handoff StrategyMeasurement CollectionSwitching Decision MakerStandard Implementation
    NCHO (Network-Controlled)Base Station measurementsMSC central switch1G Analog (Very slow, 10s latency)
    MAHO (Mobile-Assisted)Mobile phone gathers signal dataMSC switch2G GSM / 3G (Fast, 1s latency)
    MCHO (Mobile-Controlled)Mobile phone measurementsMobile phone itselfDECT / Private WLAN (Instant)
    Practical Handoff Considerations
    • Hysteresis: A margin by which the new candidate station's signal must exceed the current station's signal to trigger handover (prevents accidental swapping due to brief fading).
    • Dwell Time: The continuous duration over which a mobile remains in the boundary zone before handoff initiates (avoids sudden spikes).
    • Guard Channels: Reserving a pool of channels in each cell exclusively for incoming handovers (prevents call drops).
    • Ping-Pong Effect: Rapid, repetitive back-and-forth handovers at the cell boundary caused by fluctuations. Prevented by combining hysteresis + dwell time.
    The 4-Stage Handoff Sequence
    1
    Measure Phase: MS continuously measures downlink signal strengths (RSS) from neighbouring base stations and reports to current BSC.
    2
    Decide Phase: BSC detects that RSS falls below threshold (e.g. -100 dBm) and hysteresis is met. Initiates handoff request to MSC.
    3
    Execute Phase: MSC locates new target BS, reserves a traffic channel, and instructs current BS to send a handover command to MS.
    4
    Complete Phase: MS retunes to the new frequency, establishes link with the new BS, and old BS releases the previous channel back to the pool.

    As cellular subscriber density grows, operators must scale capacity without acquiring expensive new frequency spectrum. Three architectural scaling options are cell splitting, cell sectoring, and microzone deployment.

    CDMA Code Separation & Orthogonality

    In CDMA, every active mobile is assigned a unique Pseudo-Noise (PN) code vector from a set of **orthogonal Walsh codes**. The inner product of any two distinct codes $C_A \cdot C_B = 0$, while $C_A \cdot C_A = N$ (where $N$ is chip length). This enables multiple signals to share the same frequency band simultaneously without interfering.

    Worked Bit-Level CDMA Example

    User A has code C_A = [1, -1, 1, -1]. User B has code C_B = [1, 1, -1, -1]. Note: $C_A \cdot C_B = 1(1) - 1(1) - 1(1) + 1(1) = 0$ (orthogonal).

    Step 1
    User A wants to send bit 1 (represented as $+1$) ⇒ Transmits S_A = +1 × C_A = [1, -1, 1, -1].
    Step 2
    User B wants to send bit -1 ⇒ Transmits S_B = -1 × C_B = [-1, -1, 1, 1].
    Step 3
    Combined over-the-air signal: S = S_A + S_B = [0, -2, 2, 0].
    Step 4
    Receiver A recovers A's bit by taking the inner product of S and $C_A$:
    Result = S · C_A = 0(1) + (-2)(-1) + 2(1) + 0(-1) = 0 + 2 + 2 + 0 = +4.
    Divide by chip length N = 4: +4 / 4 = +1 (Bit 1 successfully recovered!).

    Q7Unit 3 Highlight15M
    Explain the need for spread spectrum. Contrast Direct Sequence Spread Spectrum (DSSS) and Frequency Hopping Spread Spectrum (FHSS) with a comparison table. Discuss the 5 major advantages of spread spectrum.
    Spread Spectrum Systems

    A Spread Spectrum system intentionally smears a narrow-band data signal across a much wider radio frequency bandwidth, making it resistant to jamming, eavesdropping, and multipath interference.

    DSSS vs FHSS Comparison
  • Standard Use
    • AspectDirect Sequence (DSSS)Frequency Hopping (FHSS)
    Spreading MethodMultiplies data bits by a fast Pseudo-Noise (PN) chip code (e.g. 11-chip Barker code).Hops carrier frequency rapidly across many channels according to a PN sequence.
    BandwidthConstant. Signal is spread continuously over a wide band.Narrow instantaneous bandwidth. Hops across wide band.
    Anti-JammingExcellent against wideband noise.Outstanding against selective narrow-band jamming.
    3G WCDMA / GPS / ZigBeeBluetooth / HomeRF / Military systems
    Five Major Advantages of Spread Spectrum
    • 1. Immunity to Interference: Narrow-band jamming only affects a tiny fraction of the spread signal. De-spreading at the receiver filters out jammer power as background noise.
    • 2. Low Probability of Intercept (LPI): Because power is spread over a wide band, the signal spectral density falls below the background noise floor, hiding the transmission.
    • 3. Multi-user Sharing (CDMA): Multiple users can share the same frequency band simultaneously using unique, orthogonal PN codes.
    • 4. Resistance to Multipath Fading: Wideband signals can isolate multi-path echo arrivals in time, neutralizing fading.
    • 5. High Privacy: Demodulating the signal requires knowing the exact PN sequence; without it, the transmission looks like white noise.
